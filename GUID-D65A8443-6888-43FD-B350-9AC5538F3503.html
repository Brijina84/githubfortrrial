RoundRealInt

Syntax: RoundRealInt(Real,int)

Description: Rounds the first parameter, a real number to the closest real value based on the value of the second parameter, an integer. The integer must be in the range [-14, +14] or else the function returns an unknown value.

Example: The following table shows some examples:
First Parameter (real) Second Parameter (int) Result
3.14159 -5 3.14159
3.14159 -4 3.14160
3.14159 -3 3.14200
3.14159 -2 3.14000
3.14159 -1 3.10000
3.14159 0 3.00000
