Score Models

A score model is a mathematical formula used to predict the likely behavior of a customer or prospect. Because a score model is a decision flow implementation type, a decision flow step can execute a score model directly. A scoring scenario can also execute a score model.

The Score Models list page is used to access existing score models or add new ones. The Score Model Properties page is used to define or modify score models and their properties. When defining a score model, you specify the type of calculations to be performed (additive or logistic regression), whether the score model returns reasons, and if so, whether reasons are ranked by priority (ranked reasons) or by distance from the baseline score (distanced reasons). You also create and define characteristics, which represent data (such as income or housing status) used as input values by a score model.
