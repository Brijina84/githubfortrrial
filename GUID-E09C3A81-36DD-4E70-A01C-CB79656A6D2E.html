Deleting Groups

Administrators with the proper permissions can delete the current group. However, you cannot delete a group to which you belong.
  1. In the task menu, click Security and then in the security navigation pane, click Groups.
  2. (Optional) In the search area field, enter search filter criteria and click Search.
  3. Select the group to delete and click the Delete group icon for that row.
  4. Click OK in the confirmation dialog box that appears.
