Reports

The Application Processing Module provides users with five standard reports. If needed, there is an option to create ad hoc reports and add them to the system.

When you open a report to begin your analysis, the report appears in view mode, which does not allow you to edit it. In this mode, you can use the filtering options at the top of the report page to refine your focus.

Covered Topics
  • Configuring Reports
    Administrators with the proper permissions can configure report groups and configure, modify, or delete score ranges.
  • Opening Reports
    If reports were made available to you, open them from the My Reports pane on the Application Processing Module Home page. You can also access any of the available reports using the Reports option in the task menu.
  • Application Processing Module Reports
    The Application Processing Module provides a variety of reports to assist you in evaluating the effectiveness of your system. If the role to manage reports has been assigned, you will have access to view the reports.
  • Custom Reports
    Standard report definitions are available in FICO Application Studio Designer. Use the existing reports in the Consumer, or Small Business templates in FICO Application Studio Designer to configure and create reports.
