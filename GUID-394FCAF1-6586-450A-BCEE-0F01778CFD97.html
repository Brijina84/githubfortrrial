Category Properties

The Category Properties page describes the properties associated with a product category.

When you select a category name in the categories list page, its Category Properties page appears.

The following table describes product category properties:

Property Description
Name

Name of the category. Category name is required and must be unique.

Category name can contain letters, numbers, and underscore.

Category names must also match the Application/CreditRequest/@ProductCategory attribute submitted in the XML transaction input. Communicate with your interface developer to coordinate product category values.

Description

Description of the product category.

Description is optional.

Test storage Percentage

Test storage percentage must be a whole number. The range is 0 to 100. Default is 0.

If you install the Data Storage Service (DSS), the Test storage percentage field determines the number of requests that the Data Storage Service sends to the data repository. The number of requests determines the amount of data that is available in the data repository for strategy testing using the Strategy Testing Facility (STF).

See Also For more information about the Data Storage Service, Decision Module reports, and the STF, see the FICO Origination Manager Decision Module Administrator's Guide.
Product selection tree

A product selection tree is a specialized decision tree that allows you to specify default processing for the products in a category.

  • You can specify default processing when a specific product is not requested at runtime, or when the Decision Module executes a cross-sell for a specific category.
  • You can create multiple product selection trees and select a default product selection tree from the Product Selection Tree drop-down list.
  • You can navigate to Product Selection Tree Properties page.
  • To implement very complex product selection logic, split processing logic across multiple product selection trees, and define one or more product selection trees that execute the others.
