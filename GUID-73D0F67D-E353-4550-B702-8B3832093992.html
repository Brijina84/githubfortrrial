Viewing Application Details

The Details of <Applicant Name> pane contains the following current details for the applicant and, if it is a joint application, for the co-applicant.
  • Personal Details
    • Personal ID
      Note The Personal ID field contains sensitive information. Depending on your role, you may not be able to view the information in this field.
    • Prefix
    • Name
    • Existing customer?
    • Date of birth and age
    • Marital status
    • Number of dependents
    • Identification type, number, and place issued
    • Current address (Country, State, Address city, Address, Postal code)
    • Phone number
    • Email address
  • Employment Details
    • Employment type (full time or part time)
    • Our employee (whether the applicant is employed by the organization)
    • Employer name
    • Start date
    • Months with employer
    • Annual salary
  • Financial Details
    • Total monthly gross income
    • Total monthly housing expenses
    • Check reference account
  • Assets and Liabilities
    • Total assets
    • Total monthly liabilities
  • CRA Report and Summary

    Administrators determine and configure the content in the CRA Report and Summary pane. The following items are the default fields. Your implementation may differ.

    • Summary (Includes CRA Total Monthly Payments, Max Delinquency, and Worst CRA Rating)
    • Agency Analysis (Includes Second Online CRA Score, First Online CRA Score, and Abnormal Report Indicator)
    • Inquiry Data (Includes Inquiry Count and Most Recent Inquiry)
    • Delinquency (Includes Worst Rating Reported)
    • Revolving Trade Line Data (Includes Revolving Worst Rating and Total Revolving Balance)
    • Credit File Indicators (Includes Fraud Victim, Bankruptcy Flag, and Fraud Alias)
    • Credit Report Timeline
    • Options such as View credit report summary and View credit report image.
Note For CRA field descriptions of the default fields, refer to the corresponding bureau documentation.
Covered Topics
  • Viewing the Credit Report Summary
    The View credit report summary page lists a configurable set of CRA characteristics obtained from the Data Acquisition Module.
  • Viewing the Credit Report Image
    The View credit report image displays the print image or HTML report obtained from the Data Acquisition Module. Whether the print image ot the HTML report is dispayed depends on the settings configured by your administrator. If the application has co-applicants, then the print image or HTML report for each of the applicants is displayed under their associated consumer details section.
  • Downloading the Credit Report Image
    The View credit report image displays the print image obtained from the Data Acquisition Module. You can view credit report image of an application from the CRA Report and Summary pane of the Application Overview page. You have the option to download the print or HTML image as a PDF file.
