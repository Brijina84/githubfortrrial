Creating a Data Method

Use the Create button on the Data Methods list page to add a new data method.
Before building a data method, you typically gather relevant information and create an outline of the process. For an overview of what you need to do, see Creating Data Methods: Overview.
  1. In the navigation pane, click Data Methods.
  2. On the Data Methods list page, click the Create button.
  3. In the Create dialog box, provide a name following the conventions in Data Method Naming Conventions or your own conventions.
    Names must be text strings of 150 characters or less. Names can contain letters, numbers, and underscores.
  4. (Optional) In the Description field, enter text describing what the data method does.
  5. Click Create.
    The data method is listed on the Data Methods page.

    To change any of the default properties, add input parameters and local variables, and build an expression statement, you need to edit the data method. At this point, you can proceed with the following steps to continue configuring the data method, or you can check it in to commit it to the configuration repository. If you check in the data method, you need to check it out when you are ready to make these changes.

  6. Click the data method name to display its Data Method Properties page.
  7. Click Edit.
  8. The default value for Context is the Application object. If you want the data method to be executed on a different object, click Change.
    1. In the Object Model Browser, select an object.
      (You can browse to the object or search for it.)
    2. Click Select.
  9. The default value for Result type is string. To change it, select a new value from the drop-down list.
  10. The Save history check box is selected by default. If you do not want the data method execution history to be saved, clear the check box.
    Note that while the processing time may be slightly longer when this option is selected, it can be very helpful when testing the data method.
  11. To add an input parameter for the data method, follow the steps described in Adding an Input Parameter.
  12. To add a local variable, follow the steps described in Adding a Local Variable.
  13. Build a data method expression in the Expression Editor.
    For more information on constructing expressions, see Using the Expression Editor and Using the Data Method Expression Language.
  14. Click Save.
