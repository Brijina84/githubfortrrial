Sorting the Excel Files

You can rearrange the order of the Excel files by sorting them alphabetically by the name. By default, the files are sorted in ascending order.
You must have SMW system administrator privileges to view and use the Excel Lookup Values option.
  1. In the Tools tab, select the Excel Lookup Values option.
  2. Click the Ascending icon to sort the files in ascending order or click the Descending icon to sort the files in descending order.
