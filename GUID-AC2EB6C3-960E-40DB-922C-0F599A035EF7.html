Creating a Reason Code List

Use the Create button on the Score Model Reasons list page to add a new reason code list.
  1. In the navigation pane, click Score Model Reasons.
  2. On the Score Model Reasons page, click Create.
  3. Enter a name and a meaningful description for the list.
  4. Click Create.
    The reason code list is listed on the Score Model Reasons list page.
