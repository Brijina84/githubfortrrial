Setting Template Properties

Use FICO Application Studio Manager to define the site properties for document templates, such as the maximum height and width of the images that can be uploaded to the Application Processing Module.
  1. Log on to FICO Application Studio Manager with administrator credentials and select Factory > eSpaces.
  2. In the list of eSpaces, click the link for om_apm_docgen_core.
    The Site Properties tab displays the list of settings you can configure for document templates.
  3. To have the Generate Sample button displayed on the Document Templates > Templates page of the Application Processing Module UI, click the DevelopmentMode site property and set its value to True. Click Apply.
    The default setting for DevelopmentMode is False.
  4. The FontPath site property specifies the relative path for the ARIALUNI.TTF file required for the generation of PDF documents. The default setting points to the /tmp/ location where this file is copied during the installation of the product. It is recommended that you not change this setting.
  5. To specify the maximum height for images that can be uploaded to the Application Processing Module, click the MaximumImageHeight site property and specify the height in pixels. Click Apply.
    The default value for the maximum height is 1000 pixels.
  6. To specify the maximum size for images, click the MaximumImageSize site property and specify the size in megabytes. Click Apply.
    The default value for the maximum size is 2 MB.
  7. To specify the maximum width for images, click the MaximumImageWidth site property and specify the width in pixels. Click Apply.
    The default value for the maximum width is 1200 pixels.
  8. To specify the page size for the generated documents, click the PageSize site property and enter A4 or LETTER. Click Apply.
    The default value for the page size is A4.
