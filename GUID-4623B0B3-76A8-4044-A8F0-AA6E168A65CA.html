Checking Out an Item

To edit an item in your private workspace, check it out from the configuration repository.
You will be able to check out an item if the following are true:
  • You have edit privileges for the item.
  • The item is not checked out by another user.

An item that is available to be checked out has the Check Out icon displayed. If this icon does not appear, the item is currently being used by someone else and is unavailable for you to modify, or you do not have edit privileges for the item.

Tip If you see an error message indicating that your workspace is not up-to-date, you need to update your workspace before performing any operation.
Select an item on its list page, and click the Check Out icon.

The Check In and Cancel Check Out icons are now available.
