Final Score Report

The Final Score Report provides information on the score distribution of applications, decisions, and overrides. The scores are the applicant's or the co-applicant's, based on the decision party.

This report is specific to the workflow values you have configured in your template. The following report is an example of the models used in the Consumer template. The headers represent the states of an application for that template.

To generate the Final Score Report, specify the from and to date range and score interval.

Final Score Report Fields
Field name Description
Product Type The product types configured in your system.
Score Interval The score range. You can specify a single positive value in that field. For example, if you specify 50, the score interval will be considered as follows: <50, 50-99, 100-149, and so on, depending on the number of available applications.
Applications The total number of applications for the product type that fall within the given score range.
% of Total Applications The percentage of applications in the row as compared to the total number of applications.
Approvals The total number of applications for the product type with an approved decision that fall within the given score range.
Approval Rate The percentage of approved applications in the row as compared to the total number of applications in that row.
% of Total Approvals The percentage of approved applications in the row as compared to the total number of approved applications.
Declines The total number of declined applications for the product that fall within the given score range.
Decline Rate The percentage of declined applications in the row as compared to the total number of applications in that row.
% of Total Declines The percentage of declined applications that fall within the row as compared to the total number of declined applications.
Overrides The total number of applications that:
  • Had a score below the cutoff value, but received a review decision of Approve or Approve Counter Offer, or a system decision of Approve.
  • Had a score at or above the cutoff value, but received a review decision or system decision of Decline.
Override Rate The percentage of applications with an override as compared to the total number of applications in that row.
% of Total Overrides The percentage of applications with an override as compared to the total number of overridden applications.
