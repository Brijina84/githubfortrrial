Calling a Custom Function

Custom functions are called from data methods. When constructing a data method expression, you can insert a custom function using the Insert a function icon in the Data Method Expression Editor.
In the following example, Matches is a custom function which checks for the existence of the string Decision within a string that is provided (Welcome to Decision Module). The first argument is a regular expression string (.*)Decision(.*). The function returns a string value of true or false.
