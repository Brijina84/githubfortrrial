Managing Lookup Values

In data methods, you can insert a function that operates on Microsoft Excel files. From the Data Method Expression Toolbar, you can access the Tools option to manage the lookup values. You can upload Excel files, download the Excel files, and delete the Excel files.
Note When you add an Excel file, you need to restart the Server before you can use the Excel file in data methods.
  1. Open the data method in the Edit mode.
  2. On the Data Method Expression Toolbar, click the Manage Lookup Values icon.
    The Excel Lookup Values page appears.
  3. Use the Add icon to upload an Excel file and the Delete icon to delete an Excel file.
    You can upload files with the .xlsx extension only.
  4. Click the link to navigate back to the data method page.
