Watch List

The watch list is a list of applicants that you term as risky and a decision is taken to either decline their applications or thoroughly review their applications. A watch list is configured to monitor higher risk applicants and reduce any potential loss. At application entry, APM checks the application against the entries in the watch list to check for a match. If a match is found, the application is either declined or sent for a review.

You can import and export a watch list. You can mark entries in the watch list as active or inactive.

An application is said to be a match when the following conditions match: 

(Exact match First Name AND Last name (Trimmed and Upper Case) OR
Personal ID OR
(Exact match on Address Line 1, Address Line 2, City, State and Zip for primary address) OR
Applicant Phone = Watch List Phone OR
Employer Name OR 
Email Address OR
(Email Address characters after '@') = Watch List Email Address OR
Identification Number 
AND
WatchListType = 'FraudList'
isDeleted = F
isActive = T
Covered Topics
  • Adding a Watch List Entry
    Use the Watch List > Create Watch List page to add an entry to the watch list.
  • Editing a Watch List Entry
    Administrators with the proper permissions can edit a watch list entry, if required.
  • Deleting a Watch List Entry
    Administrators with the proper permissions can delete a watch list entry.
  • Importing a Watch List
    You can import a watch list as a XLSX or XLS file. When you import a watch list, the existing list gets updated. The system checks for matching ID in the existing list. If a matching ID is found, the existing entry gets updated. If a matching ID does not exist, a new entry gets created.
  • Exporting the Watch List
    You can download the watch list as an XLS file.
