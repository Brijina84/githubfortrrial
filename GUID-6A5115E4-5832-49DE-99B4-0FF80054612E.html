Copying a Ruleset

Use the Save As icon on the Ruleset Properties page to add a new ruleset based on an existing one.
  1. On the Ruleset Properties page of the ruleset you want to copy, click the Save As icon.
  2. In the Name field, enter a unique name for the ruleset.
  3. Click Save As.
    The Ruleset Properties page of the new ruleset appears.
  4. Click Edit, and modify the description and any other properties you want to change.

    You can add or remove ruleset results, and change their order. You can also add or remove rules. If you remove a rule, a message will appear asking you to confirm the deletion by clicking Yes.

  5. Click Save.
