Testing Components and Strategies

Test components and strategies in the SMW with a base application and a test scenario.
  1. Select the Test tab in the SMW.
  2. Click Test Execution.
    The Test Execution page appears.
  3. Select a base application from the drop-down list.
  4. If a test scenario for the base application exists, select the test scenario from the drop-down list.
    The SMW displays the input test data and output test data in a grid format.
  5. If a relevant test scenario does not exist, add the input and output test data.
    1. In the Input test data tab, enter test attributes and use cases.
    2. Select a Test Attribute from the object model using the object model browser or enter a valid XPath expression.
    3. Add values for the test attributes in the Case column.
      You can evaluate multiple use cases for the test attributes.
    4. In the Output Test Data tab, add test attributes and the expected value for each use case.
    5. Repeat the steps to add test attributes and use cases to build the test data .
    6. Click the Add icon to add additional rows for test data.
  6. Click Test to run the test case.
    The Test Results appear in the same page. By default, the SMW displays the results for all the test cases.
  7. Click Show result variation to filter the results for use cases that do not match the expected output data.
  8. Click Save to save the test data and retest in the future.
