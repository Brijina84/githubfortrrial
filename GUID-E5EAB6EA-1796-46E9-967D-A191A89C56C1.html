Inserting a Function

When constructing a data method expression, you can insert functions (named procedures) to perform specific tasks.
Tip To insert a function, press Ctrl + Space at the start of a new line or after a space, and scroll down the displayed drop-down list to see the available functions; then double-click the one you want to add. Alternatively, if you know the first few characters of the name of the function, type them in, then press Ctrl + Space. All function names that start with these characters are displayed in the drop-down list. For example, typing Add and pressing Ctrl + Space displays the AddScoreModel and AddScoreModelReport functions. Double-click a function name to insert it.

To insert a function using the toolbar option, follow these steps:

  1. Click the Insert a function icon in the Expression Editor toolbar.
    In the Functions panel, the Available functions drop-drown list displays all the predefined and custom functions that are available. For each function, the name and type of arguments, and the return type are displayed.
    See Also For more information on creating custom functions and the steps to follow in order to have them included in the Available functions drop-down list, see the FICO Origination Manager Decision Module Developer's Guide.
  2. Perform one of the following actions:
    • Double-click the name of the function in the drop-down list.
    • Click the name of the function in the drop-down list, then click Insert.
  3. Close the Functions panel.
The function, with default values for its arguments, is inserted at the cursor location in the editor.
For example, the following template is inserted for the AddScoreModelReport function: 
AddScoreModelReport("",0,0,null,null)
