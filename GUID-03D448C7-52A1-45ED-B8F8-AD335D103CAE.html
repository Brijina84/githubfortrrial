Types of Errors

While using the SMW, if error messages occur they will appear in red text on the screen. The following are a few examples of instances where error messages will occur:
  • When a necessary field is blank, such as Category Name or Implementation Type.
  • When you attempt to use a name that already exists, for example a product or category name.
  • When you try to save a product that is already in use.
  • When a field contains invalid characters.

Note that the SMW does not validate entries against the domain model when you are building strategies.

See Also Error messages resulting from application decisioning are discussed in the FICO Origination Manager Decision Module Developer's Guide. Error messages are also discussed in the FICO Origination Manager Installation Guide.
