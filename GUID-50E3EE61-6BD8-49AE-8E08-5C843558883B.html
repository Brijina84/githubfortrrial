Adding Entites and Fields

You can add entities and fields to the domain model, based on your business needs
Note You can add an entity only to another entity. You can add a field only to an entity. You can identify an entity by the plus (+) sign next to it. Application is the root entity.
  1. Open the Domain Model designer or workbench.
  2. To add an entity, select the entity under which you want to add the new entity, and then click the Add Entity icon.
    The new entity is added with a default name, NewEntity with the line of business appended to the name. For example, NewEntity CL (CL for the consumer lending line of business).
  3. In the Properties pane, specify the properties for the new entity.
  4. To add a field, select the entity under whichh you want to add the field, and then click the Add Field icon.
    The new field is added with a default name, newattribute with the line of business appended to the name. For example, newattribute CL (CL for the consumer lending line of business).
  5. In the Properties pane, specify the properties for the new field.
  6. Repeat steps 2-5 to add more entities and fields.
  7. Click Save to save your changes.
  8. Click PUBLISH
