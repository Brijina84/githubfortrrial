Custom Files and Functions

Use the Custom Files and Functions option in the Tools tab to upload the JAR files that contain the custom functions to the repository, download the uploaded files, and delete uploaded files. Custom functions are called from data methods. When constructing a data method expression, you can insert custom functions. In a JAR file, define custom functions based on your business needs. When you upload the file, the functions are available for use in the data methods.

The amount of space allocated for files of Excel lookup values, external data, and custom files and functions is 150 MB per client.

The Custom Files and Functions page lists the custom JAR files and the enabled functions.

Covered Topics
  • Uploading Custom Files
    From the Custom Files and Functions option on the Tools tab, upload the custom files that contain the custom functions. Select the functions that you want to enable so that you can use them in data methods. You can uplaod one file at a time.
  • Enabling or Disabling Custom Functions
    You can enable or disable the custom functions. You can use only enabled functions in the data methods. If you no longer want to use a custom function, you can disable the function.
  • Deleting Custom Files
    You can delete the custom files that you have uploaded when you no longer need them. Ensure that the file does not contain any enabled functions.
  • Downloading the Custom Files
    You can download the custom files from the Custom Files and Functions option on the Tools tab.
  • Sorting the Custom Files and Functions
    You can rearrange the order of the custom files and functions by sorting them alphabetically by the name. By default, the files and functions are sorted in ascending order.
