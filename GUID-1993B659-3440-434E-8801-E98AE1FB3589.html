Module-level Messages

If there is a problem with tenant or system user information, the Decision Module returns an error code prefixed with FIOM- in a serviceMessage node of the Outcome/Message/MessageDetail node in the envelope.
FIOM-30000001
Error level DM_SYSTEM_ERROR
Description An unknown Decision Module system error occurred.
Resolution Check the Outcome/Message/MessageDetail element for more information. Contact your database or network administrator, or contact FICO Product Support.
FIOM-30000002
Error level DM_USER_ERROR
Description Required envelope option {1} not found. ({1} is the name of the missing option.)
Resolution Check the Outcome/Message/MessageDetail element for more information about the error and supply the missing envelope option.
FIOM-30000004
Error level DM_SYSTEM_ERROR
Description An error occurred retrieving the system user for tenant code {1} from the database. ({1} is the tenant code for which the system user could not be retrieved.)
Resolution Contact your database or network administrator to determine what is causing the data retrieval error.
FIOM-30000009
Error level DM_USER_ERROR
Description Unable to read request payload.
Resolution Check that the request payload is well-formed.
FIOM-30000010
Error level DM_SYSTEM_ERROR
Description Internal service communication error.
Resolution Check the SMW decision flows that access the internal service and confirm that the decision request properly provides for internal service input and response data.
FIOM-30000011
Error level DM_SYSTEM_ERROR
Description Unable to read internal service response.
Resolution Confirm that the decision request properly provides for internal service response data.
FIOM-30000012
Error level DM_WARNING
Description Unrecognized service name: {1} ({1} is the service name specified in the request envelope.)
Resolution Check that the internal service project is accessible to the Decision Module and that the service name is specified correctly.
FIOM-30000013
Error level DM_WARNING
Description Requested provider {1} not found. Retrying with default provider. ({1} is the provider key defined in the envelope options.)
Resolution Check that the desired provider is properly specified in Application/@ProcessingRequestType.
