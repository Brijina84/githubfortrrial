If-Then-Else Statement

An if-then-else statement is an extension of the if-then statement that provides for an action to be performed if a condition is true, and a different action to be performed if the condition is false.
An if-then-else statement has the following syntax: 
if <condition> then <statement_sequence1> else <statement_sequence2> endif
Where:
  • <condition> is any valid data method expression comparison expression (see Comparison Expressions in Conditional Statements), or any data method expression boolean identifier (local, input, data method, function, field, or value).
  • <statement_sequence1> is any valid series of statements sequence, and will execute if the <condition> evaluates to true.
  • <statement_sequence2> is any valid series of statements, and will execute if the <condition> evaluates to false.
  • Any data methods or fields used in the statement sequences are valid for the active context.
Note Field names may take the form Object.FieldName, where Object is a domain model object; or a local, input, data method, or function; or the keyword result that is an object type.

The following table contains an example of an if-then-else statement.

If-Then-Else Conditional Statement
Item Value
Name dm_FinStat_CurrentRatio
Description Returns the ratio of current assets to liabilities.
Context Application.Business.BusinessFinance.FinancialStatement
Result Type numeric
Local Variable 
Name    Data Type    Initial Value
xRatio  numeric      -9900008
Expression result := -9900008; // Missing data returns Not Answered 
xRatio := -9900008; // Special values or negative amounts return zero.
if ((TotalCurrentAssetsAmount = null) or
     (TotalCurrentLiabilitiesAmount = null)) then
  xRatio := -9900008;
else
  if ((TotalCurrentAssetsAmount <= 0) or
       (TotalCurrentLiabilitiesAmount <= 0)) then
    xRatio := 0;
  else
    xRatio := TotalCurrentAssetsAmount / TotalCurrentLiabilitiesAmount ;
  endif
endif
result := xRatio
The data method reads as follows:
  1. Assign result an initial value of -9900008, which represents a “not answered” value.
  2. Assign the value -9900008 to the local variable xRatio.
  3. If the value of either TotalCurrentAssetsAmount or TotalCurrentLiabilitiesAmount equals null, assign the value of-9900008 to xRatio, and go to step 6.
  4. If the value of either TotalCurrentAssetsAmount or TotalCurrentLiabilitiesAmount equals 0, then assign the value of 0 to xRatio.
  5. Otherwise, assign xRatio the actual value of TotalCurrentAssetsAmount divided by TotalCurrentLiabilitiesAmount.
  6. Assign result the value of xRatio.
