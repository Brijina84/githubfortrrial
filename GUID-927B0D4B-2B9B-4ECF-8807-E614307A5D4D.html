RatioINT

Syntax: RatioINT(operand1, operand2)

Description: Calculates the ratio of the operands provided based on the calculation - (operand1 / operand2).

Inputs:
  • operand1—Numeric
  • operand2—Numeric

Return Type: Numeric

Returns:
  • 0 - if the operand2 <=0
  • Numeric value based on the calculation. The output is rounded down to nearest integer.

Example: RatioINT(10,20) returns 0.
