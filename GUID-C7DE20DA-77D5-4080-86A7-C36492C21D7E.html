UCase

Syntax: UCase(theString)

Description: Converts a string to uppercase.

Inputs:

  • theString: The string being converted.

Return Type: string

Returns:

  • A copy of theString where all lowercase alphabetic characters have been changed to uppercase.
  • If theString is null, the function returns null.
  • If theString is not a valid string value, the function throws an exception.

Example: 

localString := UCase(“Text”);  // Returns “TEXT”
