Deleting Products

Administrators with the proper permissions can delete products from the system. You cannot delete a product that is associated with an application. If a product has been entered in an existing application, you must first remove the product from the credit application before deleting it.

When a product is deleted, you can no longer create an application using that product. For example, if you delete a product, it will no longer appear in the Product drop-down list on the Create Application page or in the Credit Offer Details pane on the application review page.

If you delete a product, you can create another product with the same name as the deleted product at a later time.

  1. In the administration navigation pane, click Products.
  2. (Optional) In the search area field, enter search filter criteria and click Search.
  3. Select the product to delete and click the Delete Product icon for that row.
  4. Click OK in the confirmation dialog box that appears.
