Adding Entries to the Fee Table

As an administrator with the proper permissions, use the Fees page to define all the fees you wish to make available in the Application Processing Module system. You then reference the fees you want to apply to the product in the product definition.

Fee table entries are not dependent on the scoring model suite and product categories.

  1. In the administration navigation pane, click Fees.
  2. Click Create.
  3. Enter a name for the fee table entry being added.
  4. Select a fee type for the entry.
  5. Enter a fee amount for the entry. Fee amounts must be greater than zero.
  6. In the Financed field, select Yes if the fee is financed or No if it is not.
  7. Click Create.
