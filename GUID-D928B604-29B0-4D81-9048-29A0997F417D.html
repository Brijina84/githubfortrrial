Deleting Queues

Changes in the way in which your company works may make it necessary for you to delete queues that are not needed.
After deleting a queue, you must log off and log back on to see the changes.
Note You cannot delete the Error and Search queues that come preconfigured with the system because there is no way to do so using the Application Processing Module user interface.
  1. In the administration navigation pane, click Queues.
  2. (Optional) In the search area field, enter search filter criteria and click Search.
  3. Select the check box next to the queue row to delete and click the Delete queue icon.
  4. Click OK in the confirmation dialog box that appears.
