Item Search

You can search for a specific item by entering a few characters from its name or description in the search box on the list page.

The search is dynamicâ€”results are displayed and refreshed as you type your search term. For example, if you have the data methods dm_Root_CountLiabilityBK and dmo_Root_GetCreditRequest, typing d in the search box on the Data Methods list page will display both data methods. Typing some other characters that form part of the name or description, such as root, will also list both data methods.

  • You will get the same results regardless of whether your search term includes upper case or lower case characters. In the example, searching for d or D will display both data methods.

  • Your search term can contain special characters such as ! or space, that are allowed in an item's description. For example, if the description for dm_Root_CountLiabilityBK contains the sentence "This is a test !", typing ! in the search box will display this data method in the results.

If there are no item names or descriptions that contain your search term, the list page does not display any items.

To see all the available items on the list page, make sure that the search box is empty.
