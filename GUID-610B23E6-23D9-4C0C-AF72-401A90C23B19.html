InStr

Syntax: InStr(string1, string2, boolean)

Description: Determines the 1-based position of the first occurrence of string1 in string2. If boolean is true, InStr performs a case-insensitive search. If boolean is false, InStr performs a case-sensitive search.

Note 1-based is a method for noting positions of characters in a string. In a 1-based string, the first character is considered to be at position 1. Each succeeding character is numbered one greater than the character to its left.

Inputs:

  • string1: The string to search for in string2.
  • string2: The string to be searched.
  • boolean: true indicates case insensitivity; false indicates case sensitivity.

Return Type: numeric

Returns:

  • A number representing the 1-based position of string1 within string2.
  • If string1 or string2 is null, the function returns null.
  • If string1 or string2 is not a valid string value, the function throws an exception.
  • If boolean = false, then the strings must also match on case.

Example: 

localNumeric := InStr("cde", "abcdefabcdefghij", false); // Returns 3.0, because the first time the string cde appears in the longer text string, it begins at the third character.
