Comparing Item Versions

Use the Compare option for the Version History table to compare two versions of an item.
  1. View the item's version history by performing one of the following actions:
    • On the item's list page, click the History icon for the item in the Version column.
    • Click the item's name in the list page, and then select History in the Version drop-down list.
    The Working Copy Status table displays details of the item's current working copy. The Version History table lists all previous versions of the item.
  2. In the Compare column of the Version History table, select the two versions you want to compare.
  3. Click Compare.
    The Visual Differences window appears. The two versions of the item's properties page are displayed side-by-side.
    Note The SMW may be slow to display the Properties page or the Visual Differences window when you compare versions of large decision tables such as tables with more than 50 columns and 25 rows. For optimal performance, split the decision table into smaller tables or use the pagination feature in the editor.
