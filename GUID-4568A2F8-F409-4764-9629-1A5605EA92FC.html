Document Templates

Document templates are used to define the layout for letters and other documents that are generated within the Application Processing Module.

A document template combines static text with keywords. When a document template is used to generate a letter during application processing, the keywords are replaced during the generation process with the actual values from the application being processed at the time. Note that the preconfigured Application Processing Module generates a letter when a Consumer line of business application is approved or declined. For a Small Business application, it generates a letter only when the application is declined. The user can then print the letter and mail or fax it to the customer, or send the correspondence through email.

Administrators create, edit, and delete document templates, and publish and unpublish document templates to control which templates are accessible by users. Note that any changes that are made to document templates are effective immediately in the system.

Documents generated from the document templates for the Consumer line of business applications are stored in the location specified by the DocumentTemplatesFilePath site property in the om_apm_consumer module in FICO Application Studio Manager. Documents generated from the document templates for the Small Business applications are stored in the location specified by the DocumentTemplatesFilePath site property in the om_apm_sb module in FICO Application Studio Manager.

Covered Topics
  • Viewing Document Templates
    Use the Document Templates > Templates page to access existing templates.
  • Creating Document Templates
    Creating a document template involves writing the document text as a Word document, then inserting keywords as Merge fields.
  • Modifying Document Templates
    Use the Document Templates > Edit <Template Name> page to modify templates.
  • Deleting Document Templates
    Use the Document Templates > Templates page to delete document templates that are no longer necessary.
  • Publishing and Unpublishing Document Templates
    Each document template in the Application Processing Module has a status of either Published (the template is actively used) or Unpublished (the template is inactive or waiting to be published).
  • Document Template Keywords
    Keywords can be used in document templates. The values are taken from the domain model entity attributes with no change in formatting.
  • Uploading Images
    Before anyone can use an image in a document template, the image must be uploaded to the Application Processing Module and stored.
  • Troubleshooting Display of Fonts in Letters and Documents
    Documents and letters generated during application processing sometimes do not display the Chinese and Russian characters accurately.
  • Setting Template Properties
    Use FICO Application Studio Manager to define the site properties for document templates, such as the maximum height and width of the images that can be uploaded to the Application Processing Module.
