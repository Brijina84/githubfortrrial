Data Method Expression Language

The following values and functions are used when creating data methods:
  • Object functions
  • Date and time functions
  • String functions
  • Math functions
  • List functions
  • Score Model Report integration functions
  • XPath functions
  • Microsoft Excel functions
The following special values are used in this appendix to describe what the data method functions can return:
  • Infinity—A special value for which no other value can be greater, such that infinity is greater than X for every X other than infinity itself.
  • -Infinity—A special value for which no other value can be less, such that –infinity is less than X for every X other than –infinity itself.
  • NaN—(Not a Number) A special value that indicates that a calculated numeric value is non-existent.
  • null—A special value that indicates a value has not been provided.
