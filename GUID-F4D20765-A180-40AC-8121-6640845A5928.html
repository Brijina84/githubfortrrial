Creating Domain Values

Administrators with proper permissions can add new values to existing domain values.
See Also For more information on creating new domain value types and how to display those types on the Domain Values page, see the FICO Origination Manager Application Processing Module Developer's Guide.
  1. In the administration navigation pane, click Domain Values and then click the Edit icon in the row for which you want to create a new domain value. For example, to create a new check reference account domain value, click Domain Values and then click the Edit icon in the Check Reference Account row on the Domain Values page.
  2. Click Create.
  3. Enter a name and code for the new domain value.
  4. To indicate that this is the default field or domain value, select the Is default check box.
    There can only be one default value for a field or domain value type.
  5. In the Translation area, enter a translation for the domain value for the locales that are defined in the system. For example, in the Name [ES] field, enter the domain value in Spanish; in the Name [zh-CN] field, enter the domain value in Chinese.
    Note Domain value translations do not appear for English because this value is already defined on the page. If an additional English dialect, such as British English (en-UK) is added, then it will appear to allow for translation.
  6. Click Create.
