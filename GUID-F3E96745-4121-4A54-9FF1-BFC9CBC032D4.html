User-defined Fields Created or Populated by the Decision Module

To use a user-defined field as a global variable, a data method can scan a request for some trait and then set a user-defined field to indicate presence of the trait. For example, a user-defined field might be used to flag whether the applicant is an existing customer.

Other data methods can check and update the information that appears in a user-defined field. The following sample data method checks for a UserDefinedField child of creditApplication.personalApplicant having a name of Code1 and a string data type and sets its value to input variable iValue. If no such UserDefinedField is found, one is created.

Data Method Example 1
Item Value
Data Method Name dm_App_UDF_SetCode1
Context (Path+Element) Application.creditApplication.personalApplicant
Result Type string
Input Variables 
Name     Data Type         Initial Value
iValue   string            n/a
Local Variables 
Name     Data Type         Initial Value
xUDF     UserDefinedField  null
Expression 
result := null;
if (iValue <> null) then
  // check for existing UDF with the key data.
  for every 
  Application.creditApplication
  .personalApplicant.UserDefinedField relative to
  personalApplicant do
    if ( ( UserDefinedField.Name = "Code1") and 
    ( UserDefinedField.DataType = "string") )
    then
      xUDF := UserDefinedField;
    endif
  endfor
  if (xUDF = null) then
    xUDF := New UserDefinedField;
    xUDF.Value := iValue;
    Attach (personalApplicant, xUDF)
  else
    xUDF.Value := iValue;
  endif
endif

The following Data Method Example 2 creates a new user-defined field as a child of the creditApplication.personalApplicant object. Data Method Example 3 calls the data method in Example 2 to set the value of the user-defined field with the value generated by a third data method. The name (ScoringTimestamp) and data type (timestamp) of the UserDefinedField are passed as parameters.

result := null;  
if (iValue <> null) then 
  // check for existing UDF with the key data. 
  for every 
  Application.creditApplication.personalApplicant
  .UserDefinedField relative to personalApplicant do 
    if ( ( UserDefinedField.Name = iName) and 
    ( UserDefinedField.DataType = iDataType) 
    ) then 
      xUDF := UserDefinedField; 
    endif 
  endfor 
  if (xUDF = null) then 
    xUDF := New UserDefinedField; 
	xUDF.Name := iName; 
    xUDF.DataType := iDataType; 
    xUDF.Value := iValue; 
    Attach (personalApplicant, xUDF) 
  else 
    xUDF.Value := iValue; 
  endif 
endif
