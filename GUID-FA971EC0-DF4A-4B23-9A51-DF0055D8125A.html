Deleting Custom Files

You can delete the custom files that you have uploaded when you no longer need them. Ensure that the file does not contain any enabled functions.
You must have SMW system administrator privileges to view and use the Custom Files and Functions option.
  1. In the Tools tab, select Custom Files and Functions from the left navigation pane.
  2. Click the name of the custom file that you want to delete, and then click the Delete icon.
    If the custom file contains enabled functions, you cannot delete the file.
