Enabling or Disabling Custom Functions

You can enable or disable the custom functions. You can use only enabled functions in the data methods. If you no longer want to use a custom function, you can disable the function.
  1. In the Tools tab, select Custom Files and Functions from the left navigation pane.
    The Custom Functions area lists the enabled functions.
  2. To enable a custom function, perform the following steps:
    1. Click the Show disabled custom functions link.
    2. From the list of disabled functions, select the function that you want to enable, and then click the Enable icon.
  3. To disable a custom function, perform the following steps:
    1. Click the Show enabled custom functions link.
    2. From the list of enabled functions, select the function you want to disable, and then click the Disable icon.
      Note When you disable a function that is currently being used in a data method, the custom function is not available for that data method and you must re-verify the data method.
