Ruleset Results

When you configure a rule, you can optionally indicate a candidate ruleset step result to register if the rule fires. If you define a ruleset step result for a rule, and the rule fires when its associated ruleset is executed, the ruleset step result defined for this rule is registered as a candidate step result and used by the decision flow to determine the ruleset’s actual step result. The actual step result for the ruleset determines the next step in the decision flow. If multiple rules fire suggesting multiple results, the Decision Module uses the step result with the highest precedence (as defined in the ruleset properties), or you can optionally branch the decision flow based on each ruleset step result. If you do not define a ruleset step result for a rule, the rule will not register a candidate step result value.
