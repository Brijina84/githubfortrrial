Viewing Clients

Use the Clients page to view the active clients, inactive clients, and clients whose creation is in progress. By default, the Clients page displays the active clients.
  1. Log on to the APM user interface.
  2. Click Security > Clients .
    The page lists the active clients.
  3. To view the inactive clients, click Show inactive clients.
  4. To view clients whose creation is in progress, click Show in progress.
