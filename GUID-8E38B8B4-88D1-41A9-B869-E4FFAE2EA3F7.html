Inserting Input Parameters and Local Variables

Use content assist (Ctrl + Space) to insert input parameters and local variables in a data method expression.
  1. Click the location within the Expression Editor where you want to insert the input parameter or local variable.
  2. Type in the initial characters of the name of the input parameter or local variable and press Ctrl + Space.
  3. From the drop-down list that appears, double-click the value you want to insert.
Note Data method sequences, rules, and decision tree or decision table mappings cannot use data methods that require inputs.
