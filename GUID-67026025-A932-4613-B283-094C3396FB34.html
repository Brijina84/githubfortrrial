Renaming Items

Rename an item in its properties page.

Make sure that the item you want to modify is checked out and in edit mode.

Performing a Where Used search can be helpful when you are considering renaming an item and want to see the possible effects.

  1. On the properties page of the item you want to rename, select or delete the text in the Name field.
  2. Enter a unique name.

    The name can contain alphabets, numbers, and underscores. Spaces are not allowed. Also, it is recommended that you follow the naming convention suggested for the item type.

  3. Click Save.
Note When you rename an item or a variable, run the Verify All feature to identify any validation errors. All items that reference the modified item or variable display validation errors. Reconfigure the referencing items to ensure that you can verify them successfully.
