Searching for Clients

You can search for clients that you want to view, edit, or delete. You can search by name, client number, or contract number.
  1. In the task menu, click Security > Clients.
  2. Search for a client by using one of the follwoing methods:
    • If you know the exact information of the client, enter a client name, a client code, or a contract number as the search criterion. A faster search is performed based on the explicit search criteria you provide.
    • If you do not know the exact information for a client, search for a client by entering at least one character of the client name, code, or contract number as the search criterion.
  3. Click Search.
    If any users match the search criteria, the clients will be listed in a table on the page.
  4. From the search results, select the client that you want to view, edit or delete.
