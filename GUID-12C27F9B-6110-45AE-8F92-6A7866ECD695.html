Data Method Sequences

A data method sequence is an ordered list of data methods. Data method sequences are decision flow step implementations that execute the referenced data methods in the order that they are listed. Data method sequences can also be referenced by an action node of a decision tree.

Use the Data Method Sequences list page to create data method sequences for use in one or many products.

Use the Data Method Sequence Properties page to assemble or modify a data method sequence. The Data Method Sequence Properties page contains a Manage data methods section where you construct the data method sequence by adding, modifying, or deleting data method rows. The data method order number, name, context, and data object reference are displayed in each row. The names of the data method and data object reference are links that take you to the corresponding Properties pages.

Covered Topics
  • Creating a Data Method Sequence
    Use the Create button on the Data Method Sequences list page to add a new data method sequence.
  • Adding a Data Method to a Data Method Sequence
    Use the Add icon in the Manage Data Methods section of the Data Method Sequence Properties page to add data methods to a data method sequence.
  • Rearranging the Data Methods in a Data Method Sequence
    To rearrange the data methods in a data method sequence, use the Move up and Move down arrows for the data method row.
  • Replacing a Data Method in a Data Method Sequence
    To replace a data method in a data method sequence, select a new data method for the row. If required, you will also need to modify the corresponding data object reference or context.
  • Removing a Data Method from a Data Method Sequence
    To remove a data method from a data method sequence, use the Remove current row icon in the Manage Data Methods section.
  • Copying a Data Method Sequence
    Use the Save As icon on the Data Method Sequence Properties page to add a new data method sequence based on an existing one.
  • Deleting a Data Method Sequence
    Use the Delete icon on the Data Method Sequences list page to delete data method sequences that are no longer needed.
