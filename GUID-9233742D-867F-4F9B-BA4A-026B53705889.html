Viewing Applications in Error

Use the Processing Errors option in the task menu to view erroneous applications.
Erroneous applications can be extracted using search filters that can be customized and defined on the Error page.
See Also For more information about how administrators can configure the Error page, see the FICO Origination Manager Application Processing Module Administrator's Guide.
  1. On the task menu, click Processing Errors.
  2. The Error page that opens displays information about the erroneous applications. The following list is an example, as the columns on this page are configurable:
    • Application Number
    • Applicant full name
    • Applicant unique id number
    • Applicant Home Phone
    • Application Status
    • Application received date
    • Application loan source
  3. (Optional) To open an application from the list, click the corresponding application external ID.

    Use the Application History link to view the entire processing history of the application.
