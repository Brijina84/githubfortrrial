AddScoreModelReportDetail

Syntax: AddScoreModelReportDetail(string1, string2, string3, numeric)

Description: Used in a data method in a data method sequence in a decision flow, to add a ScoreModelCharHistory element to a ScoreModelHistory element that was previously added by the AddScoreModelReport function. When stored in the DSS, this can be used in reporting.

Provides the ability to use custom model results within the standard Decision Module scoring reports.

Multiple AddScoreModelReportDetail functions can be called for the same AddScoreModelReport function. When using this function, adhere to the following:
  • Use this function after using the AddScoreModelReport function.
  • Use a common score model name for both the AddScoreModelReportDetail and AddScoreModelReport functions.
  • Test the function through STF invocation. Do not test it using data method testing.

Inputs:

  • string1: String containing the name of the score model.
  • string2: String containing the bin label.
  • string3: String containing the name of the characteristic.
  • numeric: A numeric value containing the score.

Return Type: string

Returns: "successful"

Example: For an example, see AddScoreModelReport.
