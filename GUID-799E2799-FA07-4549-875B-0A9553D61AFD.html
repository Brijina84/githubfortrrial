Creating a Product Strategy

Use the Create button in the product strategies list page to create a new product strategy.
  1. In the Product Strategies list page, click the Create button.
  2. In the Name field, enter a name.
    Names must be text strings of 150 characters or less. Names can contain letters, numbers, and underscores.
  3. Optionally, enter a description in the Description field.
  4. Click Create.
    A new product strategy appears in the list page.
  5. Select the new product strategy to view its Product Strategy Properties page.
  6. Click the Edit icon to edit product strategy properties.
  7. Click the Add (+) icon in the Manage Variables section to define variables. You can modify and use these variables in the Decision Flow Selection Tree to set conditions.
  8. Double-click the existing Unspecified node to start building the decision flow selection tree with condition nodes and action nodes.
  9. Click Save.
