Adding a Watch List Entry

Use the Watch List > Create Watch List page to add an entry to the watch list.
  1. In the administration navigation pane, click Watch List.
  2. (Optional) Select the organization.
  3. From the Decision list, select the decision that must be taken if an application matches this entry. Select Decline if the applcation must be declined or select Review if the application must be reviewed.
    If the review option is selected, the application moves to the Pending pre-bureau verification state.
  4. From the the List type, select Fraudlist as the list to which the application must move.
  5. Enter the details as required and click Create.
