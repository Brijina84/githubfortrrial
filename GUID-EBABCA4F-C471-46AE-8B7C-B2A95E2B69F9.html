Editing Applications

When an application requires edits (for example, when a credit analyst chooses to update an incomplete application), you can locate the application and edit it in the system.
Applications in any of the following states may be edited:
  • Draft
  • Submitted
  • Pending Duplicate Resolution
  • Pending Review Resolution
  • Pending Verification
  • Pending Decline Archive

Perform the following steps to edit an application:

  1. Locate and open the application.
  2. In the Application page, review information as needed.
  3. Click Edit this application.
    Applications that are not in the Approved, or the Pending Approval Archive status can be edited.
  4. Make the desired modifications to the application information.
  5. Click Submit.
    Note Be sure to confirm that all mandatory fields are entered before submitting.
  6. (Optional) Click Save if you want to save the information at this moment, and wish to submit the application at a later time.
    The state of the application will remain unchanged to other viewers until you click Save or Submit. You can also use the Exit Without Saving option.
