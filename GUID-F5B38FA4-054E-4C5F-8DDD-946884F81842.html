Using the Data Method Expression Editor

The Expression Editor is a code editor used to construct a data method expression that is evaluated when the data method is executed.

A data method expression is a series of statements. The editor's "content assist" features enable you to select and insert elements in these statements. When you press Ctrl + Space, suggestions for operators, functions, local and input variables, and object model items are displayed. When you select a suggested item such as a function or a conditional block, the appropriate code template is inserted. For the list of items with content assist, see Content Assist Features in the Expression Editor.

In addition, the Expression Editor provides:
  • Bracket matchingâ€”matching sets of brackets are highlighted to help identify the ones that are not opened or closed correctly.
  • Validation of expression statements for incorrect syntax.
  • Line numbers and line wrapping for expression statements.
  • The ability to collapse and expand sections of code (such as for loops and if-then condition blocks) using the down arrow in the gutter.
  • The ability to cut, copy, or paste text, undo or redo text, and find and replace text.
  • Editor resizing using the double-headed arrow at the bottom border of the editing pane, and F11 for full-screen mode.
    Note When the editing pane is in full-screen mode, the Expression Editor toolbar is not available.
When a new method is created, the initial statement, 
result := null;
appears in the Expression Editor by default. As you build the expression, use content assist along with the toolbar icons to insert items and templates.
Note As a programming best practice, keep your data method small, making sure that it performs just what you need it to do. Do not try to write all your rules in a single data method because this could affect performance.
Note Before you can add a data method expression, you must click the Edit icon in the Data Method page. The Expression Editor is then set in "edit" mode and its toolbar is displayed
Covered Topics
  • Content Assist Features in the Expression Editor
    When you type in text and press Ctrl + Space, the Expression Editor provides a list of suggestions based on your input. For some items, a code template is inserted when you select (double-click) the item from the list.
  • Expression Editor Toolbar Icons
    The Expression Editor toolbar contains icons for inserting operations, functions, and object model items into data method expressions, and for validating data method expression statements.
  • Inserting an Operation
    When constructing a data method expression, you can view and select simple operators such as + and -, keywords such as "new", and code templates for "if"conditions and comments.
  • Inserting a Function
    When constructing a data method expression, you can insert functions (named procedures) to perform specific tasks.
  • Inserting an Object Model Item
    Use the Object Model Browser to find, view, and select object model items when building data methods.
  • Managing Lookup Values
    In data methods, you can insert a function that operates on Microsoft Excel files. From the Data Method Expression Toolbar, you can access the Tools option to manage the lookup values. You can upload Excel files, download the Excel files, and delete the Excel files.
  • Managing Custom Functions
    You can write custom functions to implement specific business logic and use them in data methods just like other functions. From the Data Method Expression Toolbar, you can access the Tools option to manage custom functions. You can upload files that contain the custom classes or functions, delete the files, and enable or disable the custom functions.
  • Inserting Input Parameters and Local Variables
    Use content assist (Ctrl + Space) to insert input parameters and local variables in a data method expression.
  • Searching Data Method Expressions
    When building data method expressions, you can find and replace text in the Expression Editor.
  • Expression Editor Color Coding
    The Expression Editor provides color coding and formatting of text according to its use.
