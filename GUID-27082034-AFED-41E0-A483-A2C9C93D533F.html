NumStr

Syntax: NumStr(number)

Description: Converts a number to string format.

Inputs:

  • number: A number

Return Type: string

Returns:

  • A string representation of number. The returned value includes six digits to the right of the decimal point, for example, “854.123456”.
  • If number has more than six digits to the right of the decimal point, the function returns a string that has been “half-even rounded” to the sixth digit. For example, if the seventh digit is a 5, the function rounds the sixth digit to the nearest “even” number.
  • Includes special handling of number as follows:
    • If number is null, the function throws an exception.
    • If number represents infinity, the function returns “?”.
    • If number represents negative infinity, the function returns “?”.
    • If number represents the special value NaN (not a number), the function returns “?”.

Examples: 

localString:=NumStr(854.1234564); // “854.123456”
localString:=NumStr(854.1234565); // “854.123456”
localString:=NumStr(854.1234566); // “854.123457”
localString:=NumStr(854.1234574); // “854.123457”
localString:=NumStr(854.1234575); // “854.123458”
localString:=NumStr(854.1234576); // “854.123458”
localString:=NumStr(-854.1234564); // “854.123456”
localString:=NumStr(-854.1234565); // “854.123456”
localString:=NumStr(-854.1234566); // “854.123457”
localString:=NumStr(-854.1234574); // “854.123457”
localString:=NumStr(-854.1234575); // “854.123458”
localString:=NumStr(-854.1234576); // “854.123458”
