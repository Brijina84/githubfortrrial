Application Status Summary Report

Use the Application Status Summary Report to ensure that applications are not overlooked and to determine whether applications are accumulating in a particular status.

In the following example, the Application Status Summary Report shows the counts of applications in a given status for a given number of days. This report is specific to the preconfigured Small Business template because the states are specific to the workflow implemented in the Small Business template; therefore, you can use this report as an example and modify it to reflect your specific workflow. To generate the Application Status Summary Report, specify the from and to date range. You can optionally specify which product categories or products to include in the report for a given date range.

Note It is recommended that you make a copy of the preconfigured report before you modify it to reflect your workflow.
Application Status Summary Report Fields
Field name Description
Application Status The current status of the application.
Days in the Same State The number of applications that have been in a given state for the given number of days (0-1, 1-7, 7-10, 10-13, 13-31, and over 31).
Total Applications The total number of applications in a given state. This value equals the sum of all the values in the set of ages/days.
Total Requested Amount $ The total requested dollar amount associated with all applications for each status.
