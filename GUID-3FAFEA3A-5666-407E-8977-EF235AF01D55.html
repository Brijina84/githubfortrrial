Importing a Decision Tree by using an FSML File

Create a decision tree from a FICO Strategy Markup Language (FSML) file by importing the file. You can import FSML or XML files.
  1. Click IMPORT FSML.
  2. Specify a name for the decision tree that is created from the FSML or XML file that you import.
  3. Browse to the location of the file and select the PMML or XML file that you want to import, or drag and drop the file on to the drop zone.
  4. Click NEXT.
  5. Perform the following steps to map the variables in the file to the fields or data methods that exist in your object model. While mapping variables, the data type of the both variables must match.
    1. For each variable, click the object model icon to view the list of objects and references in your object model.
      The elements in the schema, along with their properties are listed under Objects and References column. When you expand a node, you can view the child elements. For each element, the Select Field or Data Method column lists all fields or data methods which have the same data type as the FSML file variable.
      Note A data method has an icon before its name to distinguish it from a field.
    2. Select the element that has the required field or data method, map the variable to either a field or a data method, and then click SELECT.
    3. Repeat Step a and b and complete the mapping for all the variables.
    You can also select a DOR
  6. Click NEXT.
  7. The Import Summary page displays a read-only report of the selections made in the previous pages. Review the Import Summary and click IMPORT.
    To change your mappings, click PREVIOUS and edit your selection.
