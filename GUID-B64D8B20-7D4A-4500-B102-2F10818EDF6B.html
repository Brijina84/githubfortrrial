Copying a Decision Tree

Use the Save As icon in the Decision Tree Properties page to create a decision tree from an existing decision tree.
  1. Select the decision tree you want to copy from the decision tree list page.
    The Decision Tree Properties page appears.
  2. Click the Save As icon to copy the decision tree.
  3. Enter an unique name for the new decision tree in the Name field.
  4. Click Save As.

    The Decision Tree Properties page of the new decision tree appears.

  5. Optionally, click Edit and modify other properties.
    Note If you delete a decision tree result, a message appears asking you to confirm deletion.
  6. Click Save.
    Note Under certain circumstances, copying a tree can cause a Process Server error to occur. To prevent this problem, carefully check mappings when you copy a tree.
