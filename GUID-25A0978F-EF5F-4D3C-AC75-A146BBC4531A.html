Customizing Search Fields

Data architects can customize the list of search filter fields by using FICO Application Studio Designer to extend entities in the om_apm_consumer, and the om_apm_sb modules for the Consumer and the Small Business templates respectively. These search filter fields then appear on the Configure Search page where users can select them to help further refine the search criteria or perform additional kinds of searches using the Filter icon. To configure search filter fields, you must have the appropriate administrative permissions.

Filter fields correspond to selected Origination Manager domain model entity attributes. For information about adding new search and queue attributes, refer to the FICO Origination Manager Application Processing Module Developer's Guide.

When you add a search filter field, you are specifying an entity attribute to search.

Search filter fields can be easily localized. To localize domain values, use the Translation area when editing domain values. To localize all other values, use the FICO Application Studio Designer.
