Score Percentile Rank Values

Reserved for future use. The Origination Manager domain model contains placeholder attributes for this information (craScoring.score PercentileRank and craScoring.scorePercentileRank CodeExternal), but the Consumer template would need to be extended to use them.

The percentile ranking for the online credit bureau score indicates the percentage of the population above which the consumer’s score ranks. The percentile is designed to help consumers understand how their credit score compares to the scores of other US consumers scored with the same model, and is provided to assist CRA subscribers in complying with the FACT Act’s Risk-based Pricing rules. Applies to Equifax US and TransUnion US only.

If a non-numeric value is returned for the score percentile rank, it appears in the score percentile rank code instead.

Format/range: 0 - 100.

Score Percentile Rank Code External Values
Value Description
N Score not provided
X Custom score requested and no record found
blank No report obtained, score not available from CRA, score not requested, or score percentile ranking not provided
Note If your CRA subscriber number is set up to request a particular score by default, a value of N will be returned if you request a different score in transaction input. However, results for the default score may still appear in the report print image.
