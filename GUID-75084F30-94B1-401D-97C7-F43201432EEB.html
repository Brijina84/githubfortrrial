Importing a Repository

You can import the configuration repository. Use the Manage Repository option in the Tools tab to import the configuration repository. If a repository already exists, reimporting the file might override the existing items or create duplicates. However, you can reimport the repository file.
Note When you import an Origination Manager 4.9.x repository into Origination Manager 5.x, ensure that XML import file contains the following schema elements:
  • Attribute "StrategyVersion" in Application (<xs:attribute name="StrategyVersion" type="xs:string"/>)
  • Attribute "TimeTaken" in ProcessingHistory(<xs:attribute name="TimeTaken" type="xs:long" use="optional"/>)
  1. In the Tools tab, select Manage Repository from the left navigation pane.
  2. In the Manage Repository page, click Import.
    If a repository already exists, click the Import link.
  3. Browse to the location of the repository file and click Open.
  4. Click Import.
    To synchronize with the configuration repository, ensure that the Check in option is enabled.
