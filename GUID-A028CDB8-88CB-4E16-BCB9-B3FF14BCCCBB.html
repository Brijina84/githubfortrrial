CRA Scoring Model Values

When requesting a consumer credit report from a CRA, a scoring model can be specified for online credit bureau score 1 and online credit bureau score 2. These values can appear in the following places:
  • The CRAScoringModel.ModelNumber attribute in the om_apm_cra_core module in Designer.
  • The CRAScoringModelStructure.ModelNumber attribute in the om_apm_bootstrap module in Designer.
  • The CRA Configuration > Edit CRA page in the Application Processing Module user interface.

In the Score 1 column, an X denotes a scoring model that is acceptable as score 1 for transactions using the ARM 3.0 model suite.
