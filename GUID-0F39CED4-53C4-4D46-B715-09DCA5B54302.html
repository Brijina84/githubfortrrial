Changing and Resetting Passwords

You can reset a password when a user is locked out of the system, has forgotten the password, or if the password expires. Password management is provided by the access management system connected to the application.
Note The password can be sent to the user through email only if your installation is configured to allow correspondence through email. For more information about passwords, LDAP, TDS, or OpenAM see the FICO Origination Manager Installation Guide.
Note Security administrators with the proper permissions can also use the access management system consoles to reset a password.
  1. To send a link to reset password, click Security > User Passwords.
  2. Select the user name.
    The Username and Email address is displayed.
  3. Click Send Password Reset Link
    An email is sent to the email address of the user with instructions to reset password.
  4. As a user you can also use Change Password option from the task menu.
    An email is sent to the email address of the user with instructions to reset password.
