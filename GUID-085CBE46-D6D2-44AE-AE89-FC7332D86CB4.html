Creating a Score Model from PMML File

Create a score model by importing a scorecard PMML file to augment your decision logic.
Scorecard PMML files can contain reason code lists. When reason code lists are detected during the PMML import, a reason code list is automatically generated.
Note PMML files created by the Model Builder only can be imported. Model Builder is a product for developing predictive models. Model Builder produces PMML that conforms to the Mining model specification.
  1. Click Import PMML.
  2. Specify a name for the score model that is created from the PMML file that you import.
  3. Browse to the location of the file and select the PMML file that you want to import, or drag and drop the file on to the drop zone.
  4. Click NEXT.
  5. Map the variables in the file to the variables that exist in your object model. Perform the following steps:
    1. For each variable, click the object model icon to view the list of objects and references in your object model.
      The node elements in the schema, along with their child elements are listed under Objects and References column. When you expand a node, you can view the child elements. For each element, the Select Field or Data Method column lists all fields or data methods which have the same data type as the FSML file variable.
      Note A data method has an icon before its name to distinguish it from a field.
    2. Select the element that has the required field or data method, map the variable to either a field or a data method, and then click SELECT.
    3. Repeat Step a and b and complete the mapping for all the variables.
    You can also select a DOR
  6. Click NEXT.
  7. The Import Summary page displays a read-only report of the selections made in the previous pages. Review the Import Summary and click IMPORT.
    To change your mappings, click PREVIOUS and edit your selection.
The scorecard and reason code list are automatically checked in. You will need to check them out first in order to edit them. The reason code list is also added to the Score Model Reasons list.
