Promoting an Item Version

The working copy of an item is the version that is currently active and editable when checked out. You can promote an earlier version of the item to be the working copy. This feature can be helpful if you make changes to an item and later find that you need to revert to the previous version.

Before promoting another version to working copy status, check in recent edits to make sure they are not lost.

  1. If the item is not checked out already, select it on the list page and click Check Out.
  2. Click the item's name.
    The Version drop-down list appears. The list displays Working Copy, History, and the various version numbers for the item.
  3. Select the version you want to promote.
  4. Click the Promote icon in the actions toolbar.
  5. Click Yes when prompted whether you want the version that is checked out to be overwritten.
    The selected version is now the item's working copy.
    Note The Decision Module requires items to have unique names. But you can accidentally create items with identical names when you promote an earlier version of an item with the same name as an existing item. For example, rename an item from A to B. Create another item with the name A. Check out item B and promote the version with the name A.
