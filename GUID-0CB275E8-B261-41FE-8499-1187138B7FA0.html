Creating a Score Model

Use the Create button on the Score Models list page to add a new score model.
  1. In the navigation pane, click Score Models.
  2. On the Score Models list page, click the Create button.
  3. In the Create dialog box, provide a name for the score model.
    Names must be text strings of 150 characters or less. Names can contain letters, numbers, and underscores.
  4. (Optional) In the Description field, enter text describing the score model.
  5. Click Create.
    The score model is listed on the Score Models page.

    To specify the properties of the score model and add variables and characteristics, you need to edit the score model. At this point, you can proceed with the following steps to continue configuring the score model, or you can check it in to commit it to the configuration repository. If you check in the score model, you need to check it out when you are ready to make changes.

  6. Click the score model's name to display its Score Model Properties page.
  7. Click Edit.
  8. The default value for Calculation type is Additive. Retain this value or change it to Logistic Regression depending on how you want the aligned score calculated.
    • Additive—partial scores for each of the individual scored characteristics are added together to produce a raw score, and this raw score value becomes the adjusted score. Alignment calculations are then applied to the adjusted score to produce an aligned score.
    • Logistic Regression—partial scores for each of the individual scored characteristics are added together to produce a raw score, then a fixed adjustment amount is added to produce an adjusted score, and a regression formula is applied to the adjusted score. Alignment calculations are then applied to the adjusted score to produce an aligned score.
  9. In the Minimum field for Range, enter an integer representing the lowest score value that may be returned by the model when it is executed.

    If the calculated value is less than this value, the final score is set to this value.

  10. In the Maximum field, enter an integer representing the highest score value that may be returned by the model when the model is executed.

    If the calculated value is greater than this value, the final score is set to this value.

    Note The Maximum and Minimum fields are interlinked. If the minimum value is greater than the maximum value or if you specify only the minimum value and do not specify the maximum value, the system displays an error accordingly. In such a scenario, when you specify the maximum value, the minimum field value changes to 0. You must again enter the minimum value.
  11. In the Multiplier field for Alignment, enter a real number.

    This value is used in conjunction with the alignment factor in the next step to produce an aligned (final) score according to the formula: aligned score = (adjusted score * alignment multiplier) + alignment factor

    Consult your model developer to obtain the values for the alignment multiplier and alignment factor.

  12. In the factor field, enter an integer.
  13. In the Slope field for Log odds, enter a real number.

    This value is used in conjunction with the log odds intercept in the next step to calculate the odds from the score value. The values to be entered into these fields are determined by modelers during model development.

  14. In the intercept field, enter a real number.
  15. In the Reason codes list field, select the reason code list.
  16. In the Manage Variables section, add variables and map them to the object model fields to be used in defining characteristics.
    For more information on setting up variables, see Creating and Mapping Score Model Variables.
  17. Click Save.
After you have created and defined the score model, add characteristics and their bins for the model type you selected.
