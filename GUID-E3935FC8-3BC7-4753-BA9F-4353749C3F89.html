Copying a Product Category

Use the Save As icon in the Category Properties page to create a new product category from an existing product category.
  1. Select the category you want to copy from the categories list page.
    The Category Properties page appears.
  2. Click the Save As icon.
  3. Enter an unique name for the new product category in the Name field.
  4. Click Save As.
    The Category Properties page of the new category appears.
  5. Click Edit, and modify the description and any other properties you want to change.
  6. Click Save.

The product category is added to your private workspace. You must check in the product category to commit it to the configuration repository.
