Data Method Types

Data methods can be grouped into three categories, or types, depending on the kind of work they do:
  • Data methods that return values
  • Data methods that return domain model objects
  • Data methods that perform work, but do not return anything

Each data method type has a recommended naming convention and is best suited for certain usages.
