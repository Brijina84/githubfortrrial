TimestampToTime

Syntax: TimestampToTime(timestamp)

Description: Returns the time portion of a timestamp.

Inputs:

  • timestamp: A timestamp.

Return Type: time

Returns: The time portion of timestamp. If timestamp is null, the function returns null.

Example: 

localTime := TimestampToTime(Now());
