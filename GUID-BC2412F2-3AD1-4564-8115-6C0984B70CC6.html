Adding a Data Method to a Data Method Sequence

Use the Add icon in the Manage Data Methods section of the Data Method Sequence Properties page to add data methods to a data method sequence.
Note Data methods that have inputs cannot be used in data method sequences.
  1. In the Manage Data Methods section, click the Add icon.
    A row for specifying a data method is displayed.
  2. Add a data method using one of these options:
    • If you know the name of the data method, start entering it in the Data Method cell in the new row. From the drop-down list, select a value.
    • Click the Open object model browser icon in the Data Method cell in the new row, select a data method from the Object Model Browser, and click Select. If you do not know where in the object model the data method is defined, use the Search feature to locate it.
    The name of the selected data method is displayed in the Data Method cell and its context path is displayed in the Context cell. The same context path is displayed in the Data Object Reference cell for this row.
  3. If you want to run the data method only for a subset of the objects in the context path, select a data object reference from the drop-down list in the Data Object Reference cell.
    The data object reference name replaces the context path that was displayed in the cell.
  4. Repeat steps 1â€“3 for each additional data method you want to include in the data method sequence.
  5. Click Save.
    Note
    • It is possible to add multiple invocations of the same data method to the data method sequence.
    • Data method rows cannot be copied.
