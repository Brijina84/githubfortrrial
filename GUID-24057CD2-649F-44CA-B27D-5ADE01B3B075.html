Deleting Excel Lookup Files

Use the Delete icon on the Excel Lookup Values page to delete the Excel files that you no longer need. You cannot delete a file if any data methods use the file.
You must have SMW system administrator privileges to view and use the Excel Lookup Values option.
  1. In the Excel Lookup Values page, click the Delete icon.
  2. Select the file that you want to delete and click Delete.
  3. When prompted, click Yes to confirm.
    Note If the Excel file is used in a data method, the Excel Lookup File in Use information dialog box opens. The box lists the data methods that use the Excel file. If a built-in function uses the Excel file, then you have to remove the file from the function, before you can delete the file. However, if a custom function uses the file, you are able to delete the file directly.
