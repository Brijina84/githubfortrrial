Content Assist Features in the Expression Editor

When you type in text and press Ctrl + Space, the Expression Editor provides a list of suggestions based on your input. For some items, a code template is inserted when you select (double-click) the item from the list.
The Expression Editor provides content assist for the following items:
  • Data Method Expression Language keywords
  • Operators
  • Object model data
  • Functions
  • Local and input variables
  • Data methods
  • Data object referencesâ€”a syntactically correct for loop with default values
  • Child objects of the data object that is the context for the current data method
  • if-then and if-then-else condition blocks
  • Inline and block comments
  • Code templates for custom functions and internal services
