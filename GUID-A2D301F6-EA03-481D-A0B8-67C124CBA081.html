Downloading Excel Files

You can download the Excel files that contain the lookup values from the Excel Lookup Values option on the Tools tab.
You must have SMW system administrator privileges to view and use the Excel Lookup Values option.
  1. In the Tools tab, select Excel Lookup Values from the left navigation pane.
  2. Click the name of the file that you want to download.
    Note File download fails if the download location contains a file with the same name and extension. In such a scenario, it is recommended that you either delete the existing file before downloading or save the file to a different location.
    The file is downloaded to your local machine.
