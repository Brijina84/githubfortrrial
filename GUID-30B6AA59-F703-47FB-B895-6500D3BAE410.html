How-To Task Library

Library of all tasks that you perform.
Application Processing - Administrative Tasks

Perform administrative tasks such as defining decision strategy parameters, adding and maintaining clients, users and user groups, printing and reviewing reports, and monitoring daily application processing activities. Security and system administrators would perform these tasks.

Application Processing - User Tasks

Learn how to create applications, enter and update application data, obtain credit report and scoring data, and process applications through the system. Data entry clerks, credit analysts, customer service representatives, supervisors, and managers would perform these tasks.
