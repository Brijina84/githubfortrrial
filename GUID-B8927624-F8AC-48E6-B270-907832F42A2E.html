Logistic Regression Calculations

In logistic regression calculations, the adjusted score of the model is obtained by applying an adjustment amount and a regression formula to the raw score. The adjusted score is then used in the process for determining the final score. If you select the logistic regression calculation type option, the following steps are performed to calculate the final score.
  1. First, the model sums all partial scores assigned to a characteristic to produce a raw score.
  2. Next, the adjusted score is calculated by applying an adjustment amount (assigned by modelers during model development and specified in the Score Model Properties page when logistic regression is selected) to the raw score and then applying a regression formula: 
    adjusted score = 1000 * e**(raw score + adjustment amount) / (1+e**(raw score + adjustment amount))

    where e is the constant 2.7182818285 (the base of the natural logarithm).

  3. Then an aligned score is calculated: 
    aligned score = adjusted score*m+f

    where m and f are the alignment multiplier and factor respectively, as assigned by the model developer and specified in the Score Model Properties page.

  4. The final score is the final result of a score model, to be used in subsequent processing. The final score is determined as follows:
    • If the aligned score is below the minimum score value assigned by the model developer, final score is the score model’s minimum score value.
    • If the aligned score is above the maximum score value assigned by the model developer, final score is the score model’s maximum score value.
    • Otherwise, final score is set to the aligned score.
