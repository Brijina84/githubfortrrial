Creating Custom Functions

The Decision Module allows you to write custom functions to implement specific business logic and use them in data methods just like other functions.

A custom function can be a simple utility function or a complex function such as one which invokes a web service or performs a database query. Custom functions are invoked during rule execution at runtime.

Covered Topics
  • Calling a Custom Function
    Custom functions are called from data methods. When constructing a data method expression, you can insert a custom function using the Insert a function icon in the Data Method Expression Editor.
