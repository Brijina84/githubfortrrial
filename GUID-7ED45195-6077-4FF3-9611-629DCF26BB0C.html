If-Then Statement

Use an if-then statement when you want a specific action (or actions) to be performed if a condition is true. An expression can have one or multiple if-then statements.
An if-then statement has the following syntax: 
if <condition> then <statement_sequence> endif
Where:
  • <condition> is any valid boolean identifier (local, input, data method, function, field, or value) or comparison expression (see Comparison Expressions in Conditional Statements).
  • <statement_sequence> is any valid series of statements.
  • Any data methods or fields used in statement_sequence are valid for the active context.
Note Field names may take the form Object.FieldName, where Object is a domain model object; or a local, input, data method, or function; or the keyword result that is an object type.

The data method in the following table contains an example of an if-then statement.

If-Then Conditional Statement
Item Value
Name dm_Root_AutoLoantoValueRatio
Description The requested AdvanceAmount for the loan divided by the ValueAmount of the collateral, where the collateral type is “auto”.
Context Application
Result Type numeric
Local Variables 
Name             Data Type          Initial Value
xCreditRequest   CreditRequest      null
xCollValue       numeric            0.0
xRatio           numeric            0.0
Expression result := 0; 
xCollValue := dm_Root_TotalAutoCollateralValue;
xCreditRequest := dmo_Root_getCreditRequest;
if (xCollValue <> 0) then
  xRatio := xCreditRequest.AdvanceAmount / xCollValue;
endif
result := xRatio
The data method reads as follows:
  1. Assign result an initial value of 0.
  2. Assign the value produced by the data method dm_Root_TotalAutoCollateralValue to the local variable xCollValue.
  3. Assign the object produced by the data method dmo_Root_getCreditRequest to the local variable xCreditRequest.
  4. If xCollValue does not equal 0, then divide the credit request advance amount by the value of xCollValue, and store that value in the local variable xRatio.
