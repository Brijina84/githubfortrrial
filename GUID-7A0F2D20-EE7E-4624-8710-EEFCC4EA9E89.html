Origination Manager Overview

FICO® Origination Manager is a multi-module solution with preconfigured components that you can use to create an end-to-end application processing solution based on your company's requirements.

Some of the key features of Origination Manager are:

  • Configurable user interface
  • Domain model extension
  • Multi-tenancy
  • User roles and queue management
  • Access to scoring models
  • Access to third-party data sources
  • Integrated analytics, strategy testing, and reporting
  • Document generation capabilities
  • Service-oriented architecture (SOA) technology platform
  • Enabled for localization

Origination Manager consists of four modules:

  • Application Processing Module (APM): Enables end-to-end application processing, user queue management, preconfigured workflow processing steps (which can call the other Origination Manager modules for data and analytics), and reports.
  • Decision Module (DM): Enables the development, testing, and deployment of business rules, strategies, and custom models.
  • Data Acquisition Module (DAM): Provides access to FICO® scores as well as other consumer and business credit report data from credit reporting agencies (CRAs).
  • Analytic Module (AM): Provides access to FICO® pooled models. Accepts application and consumer credit report data and produces FICO®Small Business Scoring ServiceSM (SBSSSM) scores, and scoring reasons.

All modules function as standalone products; however, when used together, they provide a unified solution to meet your business needs. The Origination Manager solution allows the client organization to realize greater profitability by booking more loans, accurately assessing credit risks, and reducing operational costs and delays through deployment of consistent lending strategies and predictive analytics.
