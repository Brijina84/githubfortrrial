Auditing Strategy Changes

You can audit business strategies by generating a list of changes made to the configured components within a specific period.

Use this report to track and verify changes before moving your strategies to production. The report is generated from the items in your private workspace as well as the items that have been checked into the configuration repository by Strategy Management Workstation users. To make sure that the latest versions of all items are included in the report, it is important that you update your private workspace before generating the report.

The Audit Strategies option is available in the left navigation pane of the Tools tab to users who have auditing permissions.
See Also For more information on the predefined user roles that have auditing permissions, see the FICO Origination Manager Decision Module Administrator's Guide.
Covered Topics
  • Viewing a List of Strategy Changes
    Use the View button on the Audit Strategies page to get a report of item changes that were made within a specific period.
  • Strategy Change Details
    When you audit strategies, the list of changes generated includes information about the items that are checked into the configuration repository as well as the items that exist in your private workspace.
