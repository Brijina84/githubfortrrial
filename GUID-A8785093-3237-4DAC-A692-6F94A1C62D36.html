Copying a Data Method Sequence

Use the Save As icon on the Data Method Sequence Properties page to add a new data method sequence based on an existing one.
  1. In the Data Method Sequence Properties page of the data method sequence you want to copy, click Save As.
  2. In the Name field, enter a unique name for the data method sequence.

    Names must be text strings, and contain no spaces or special characters except underscores.

  3. Click Save As.

    The Data Method Sequence Properties page of the new data method sequence appears.

  4. Click Edit, and make your changes to the description and data method rows.
  5. Click Save.
