Working With Product Categories

Product categories are groups of related products, such as a loan, credit card, or other service products provided by an institution. You can define product categories from Categories and Products in the navigation pane.
The categories list page appears when you select Categories and Products from the navigation pane. You can perform the following actions in the category list page:
  • Create categories
  • Navigate to category properties
  • Preview category properties
  • Delete categories
  • Check in and check out categories

When you select a category name, its Category Properties page appears. The Category Properties page displays the category properties associated with a category. You can also view the product selection trees and products associated with a category.

Note If your system includes both the Decision Module and the Application Processing Module, the Application Processing Module makes a call to the Decision Module by submitting request XML that specifies the category and product to use for processing an application.

You can also version categories using the Check Out and Check In icons in the Category Properties page.

Covered Topics
  • Category Properties
    The Category Properties page describes the properties associated with a product category.
  • Creating a Product Category
    Use the Create button in the Categories list page to create a product category.
  • Copying a Product Category
    Use the Save As icon in the Category Properties page to create a new product category from an existing product category.
  • Deleting a Product Category
    Use the Delete icon in the categories list page to delete a product category that is not required.
  • Using Product Selection Trees
    Product selection trees are special decision trees that specify which product should be processed in specific circumstances. Add product selection trees in the Categories Properties page.
