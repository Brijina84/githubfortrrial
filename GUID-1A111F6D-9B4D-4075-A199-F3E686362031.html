Creating a JAR File for the Custom Function

When you have completed the implementation of your custom function, create a JAR file.
  1. Generate the JAR file using tools of your choice.
  2. Ensure that the JAR file is uploaded from the SMW user interface.
    For more information about how to upload the JAR files for custom functions, see the FICO Origination Manager Decision Module User's Guide.

After the JAR file is uploaded, repackage the WAR files and redeploy them.
