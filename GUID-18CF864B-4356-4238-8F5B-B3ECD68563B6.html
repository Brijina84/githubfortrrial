The IDAJavaExtensionDispatcher Interface

The com.fairisaac.eos.da.extensions.javaext. IDAJavaExtensionDispatcher interface provides for the declaration of the methods in the following table.
Method Name Description
init

This method is used to initialize predefined values or other functionality which is to be performed before the custom function is executed. It is called only once, before any other function is invoked. It does not have arguments or a return type.

Reload

This method reinitializes values and is called when reloading is performed. It does not have arguments or a return type.

isError This method checks to see whether errors occurred during an operation. It returns true in case of an error. You can write your own algorithm for error handling.
getLastError

If errors occurred during an operation, this method returns the last error as a value of type string. You can write your own algorithm for error handling.

resetLastError

If an error occurred during an operation, this method resets the error. You can write your own algorithm for error handling.

getFunctionInfo
This method defines and returns a custom function's name, description, and data type. You can define any number of custom functions in this method. Here is an example: 
DAJavaCollection customFunctionCollection = new
DAJavaCollection(); 

customFunctionCollection.addDAJavaElement(new
DAJavaElement("<Function Name>", "<Function Data
Type>", "<Function Description>"));
This method returns the DAJavaCollection object defined in the IDAJavaExtensionDispatcher interface. The object contains a list of custom functions that have been defined.
getArgumentInfo
This method defines and returns the argument's name, description, and data type for the input custom function name. You can define any number of arguments for each custom function in the method. Here is an example for defining custom function arguments: 
 DAJavaCollection customFunction = new
DAJavaCollection();

customFunction.addDAJavaElement(new
DAJavaElement("<Argument Name>", "<Argument Data
Type>", "<Argument Description>"));
This method has one argument, functionName, which is a string containing the name of the function for which argument information is required. It returns the DAJavaCollection object defined in the IDAJavaExtensionDispatcher interface. The object contains a list of arguments defined for the input function name.
executeFunction
This method executes the function based on the index that matches the input name. The implementation is based on user requirements and has two arguments:
  • functionName— a string containing the name of the function to be executed
  • functionArguments— a vector containing the list of arguments based on the index
This method returns a value of type Object.
The IDAJavaExtensionDispatcher class is implemented as a single instance. DAJavaextInvoker invokes the executeFunction method to execute user-defined functionality.

Example: IDAJavaExtensionDispatcher Implementation

package com.fico.custom.example;

import java.util.HashMap;
import java.util.Vector;
import com.fairisaac.eos.da.extensions.javaext.IDAJavaExtensionDispatcher;

public class WriteCustomFunctionsExample implements IDAJavaExtensionDispatcher{

	/*
	* Initialize values. Called only once before any other function is invoked.
	*/
	public void Init(){}

	/*
	* Called anytime a reload is performed.
	*/
	public void Reload(){}

	/*
	* Returns true if an error occurs.
	*/
	public boolean isError(){
    	    return false;
    	}

	/*
	* Returns the last error if one occurred.
	*/
	public String getLastError(){
        	    return "";
    	}

	/*
	* Resets the last error if one occurred.
	*/
	public void resetLastError(){}

	/*
	 * Defines the custom functions names with arguments and return data-types.
	 */
    public DAJavaCollection getFunctionInfo() {
		DAJavaCollection functionsCollection = new DAJavaCollection();;
        	functionsCollection.addDAJavaElement(new DAJavaElement("customFunction", "string", 
"Custom Function Description"));
		return functionsCollection;
    	}

	/*
	 * Defines the custom functions arguments data type and description information .
	 */	    
    public DAJavaCollection getArgumentInfo(String functionName){
		HashMap functionArguments = new HashMap();

		DAJavaCollection customFunction = new DAJavaCollection();
customFunction.addDAJavaElement(new DAJavaElement("firstArgument", "string", 
"First Argument Description"));
customFunction.addDAJavaElement(new DAJavaElement("secondArgument", "string",
 "Second Argument Description"));
				
		functionArguments.put("customFunction", customFunction);

		return (DAJavaCollection) functionArguments.get(functionName);
    	}


	/*
	 * Execute the function based on the index.
 	 */
public Object executeFunction(String functionName, Vector functionArguments){

		if (functionName.equalsIgnoreCase("customFunction")){
			
            return functionArguments.get(0).toString() + functionArguments.get(1).toString();
        		}
        return "";
    	}

}
