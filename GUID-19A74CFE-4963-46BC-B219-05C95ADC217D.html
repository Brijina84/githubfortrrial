Clients Page

The Clients page lists all the clients that you have configured.

You can view all clients in the system, view inactive clients, search for a client, and deactivate clients. You can track the client creation. You can view and edit client details.

By default, the Clients page contains the following tabs:
Clients page tabs 
Tab
 
Description
Show Active Clients Lists all active clients.
Show Inactive Clients Lists all inactive clients.
Show In Progress List all clients whose creation is in progress. Once the client creation is complete, the client is active and is listed in the active clients list.

Use the Activate and Deactivate command buttons to activate and deactivate clients.

Covered Topics
