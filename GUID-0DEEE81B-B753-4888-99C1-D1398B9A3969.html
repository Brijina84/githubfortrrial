Timestamp

Syntax: Timestamp(date,time)

Description: Converts a date and time to a timestamp, for use in timestamp operations.

Inputs:

  • date: An optional date.
  • time: An optional time.

Return Type: timestamp

Returns:

  • A timestamp if successful, otherwise null.
  • If date is null, the function uses 1199-11-11.
  • If time is null, the function uses midnight (00:00:00).
  • If both date and time are null, the function returns 1199-11-11 00:00:00.
  • If either date or time is invalid, the function throws an exception.

Example: 

localTimestamp := Timestamp(DuplicateApplicationLimitDate,null);
