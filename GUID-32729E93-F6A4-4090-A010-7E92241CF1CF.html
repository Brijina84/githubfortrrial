Copying a Decision Table

Use the Save As icon in the Decision Table Properties page to create a decision table from an existing decision table.
  1. In the Decison Table Properties page, click the Save As icon.
  2. In the Name field, enter a name for the new decision table.

    Names must be text strings of 50 characters or less and can contain letter, numbers, and underscores.

  3. Click Save As.

    The Decision Table Properties page for the new decision table appears.

  4. Click Edit, and modify the description and any other properties you want to change.
  5. Click Save.
    Note Under certain circumstances, copying a table can cause a Process Server error to occur. To prevent this problem, carefully check mappings when you copy a table.
