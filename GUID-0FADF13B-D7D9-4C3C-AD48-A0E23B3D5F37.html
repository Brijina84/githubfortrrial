Product Check Out, Cancel Check out and Check In

The SMW enables multiple users to work on products by allowing you to check out, check in and cancel checkout of products.

You create new versions of existing products each time you check out and check in a product.

The SMW provides the two options to manage products:
  • You can use the Check Out, Cancel Check Out, and Check In icons in the Products section from the Category Properties page.
    Note You must check out the category and edit the category.
  • You can use the Check Out, Cancel Check Out, and Check In icons in the Product Properties page.

The following topics describe the steps to check out, cancel checkout, and check in products from the Product Properties page.

Covered Topics

  • Canceling a Product Check Out
    Use the Cancel Check Out icon in the Product Properties page to undo a product check out.
  • Checking In a Product
    Use the Check In icon from Product Properties page to check in a product.
