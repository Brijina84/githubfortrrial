Duration

Syntax: Duration(value, units)

Description: Creates a duration of the specified value and units.

Inputs:

  • value: An integer indicating the duration’s value.
  • units: A string that represents the units of measure of the value. It can be any of the following: years, months, days, or milliseconds.

Return Type: duration

Returns:

  • A duration derived from the value and units passed.
  • If either value or units is null, or if units is not a valid duration units specifier, the function returns null and also throws an exception.
  • If value is not a valid numeric value, the function throws an exception.

Example: 

localDuration := Duration(6,”days”);    //Returns 6 days
