Strategy Component Snapshot Details

The strategy component snapshot for the current date includes information about the items that are checked into the configuration repository as well as the items that exist in the workspace of the user generating the snapshot. The snapshot for an earlier date only includes information about the items that were checked into the configuration repository as of that date.
In the snapshot, item types are listed in the order in which they appear in the left navigation pane of the Item Types tab. For individual items within each item type, the following details are displayed:
  • The latest version number of the item. "New" indicates that the item exists only in the private workspace of the user and has not been checked into the configuration repository.
  • The date the item was last checked in.
  • The comment that was entered when the item was last checked in.
  • The user who checked in the item.
  • Details from the item's properties page.

Deleted Items

Items that have been deleted from the configuration repository are not displayed in the snapshot. For example, if an item was present in the repository on an earlier date, but permanently deleted on the current date, the snapshot for either date does not display the item. Items that are deleted only from the user's workspace (and displayed in strikethrough format on the item's list page), are displayed in the snapshot.

Navigation

You can navigate between different sections of the snapshot using the links for item names. For example, the section for the Revolving category in the above figures contains a link for the product PT3T. Clicking this link will take you to the section that contains the details of the product.
