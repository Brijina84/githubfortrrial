Viewing Document Templates

Use the Document Templates > Templates page to access existing templates.

You can view, edit, publish, or unpublish existing document templates. You can also create and delete document templates from this page.

The Document Templates > Templates page includes the following information for each document template:

  • Name of the document template
  • Description of the document template (Click the icon near the document template name to view descriptions)
  • Date the document template was created and last modified
  • Status (active or inactive) of the document template

To view document templates:

  1. In the administration navigation pane, click Document Templates > Templates.
  2. (Optional) In the search area field, enter search filter criteria and click Search.
  3. Select the document template to view and click the Edit icon for that row.
  4. Click Preview. When you are finished viewing the document template, click Close.
