List Pages

A list page is displayed when you click an item type such as Decision Flows or Data Methods in the navigation pane. All list pages contain a Create button, a text box for search, a panel of action icons, and a table listing the items available for viewing and selection.
The following item types have list pages:
  • Categories and Products
  • Product Strategies
  • Decision Flows
  • Data Method Sequences
  • Data Methods
  • Data Object References
  • Rulesets
  • Decision Tables
  • Score Models
  • Decision Trees
  • Scenarios

You can add new items of the same type using the Create button.

You can search for a specific item by entering a few characters from its name or description in the search box. For more information on how search works, see Item Search.

When you select an item in the table, you can perform the following actions depending on your permissions and the icons that are available and enabled:

For each item in the table, the name, description, status, owner (if the item is checked in), active version, and the date of last modification are displayed.

The name of an item is a link. When you click an item name, you are taken to a page that displays the item's details and provides icons for additional actions such as Edit.

Status shows the current read or write status of the item depending on its current action state.
