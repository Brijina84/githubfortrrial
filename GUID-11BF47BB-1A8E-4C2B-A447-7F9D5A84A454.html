Where Used Search

Use Where Used search to find where items are used in your business strategies. Finding where an item is used can be helpful if you are considering deleting or renaming an item and want to see the possible effects. Also, if the search shows that an item is not used anywhere, it could indicate a possible problem.
When a Where Used search completes, the results are displayed in the Where Used dialog box for the item. The following information is provided:
  • Item types that use the item
  • For each item type, the referencing items
  • For each referencing item, a link to its properties page

Variables that are mapped to a data method or a data object reference are listed for the referencing item. Product selection trees and rules that reference an item are also named.

Covered Topics
  • Locating Where Used Items
    Use the Where Used option on an item's properties page to find where the item is being referenced.
  • Where Used Search Reference
    The Where Used Reference table shows where items of a specific type can be used in the Decision Module.
