Score Models That Return Ranked Reasons

Choose this option when you want to return reason codes based on their priority order in the reason code list, which is a preconfigured set of 100 reason codes, ranked from 00 to 99. Depending on the reason code priorities set by your organization, the most important reason codes appear near the top of the list. When reason codes are returned, you see the topmost ten reasons.
See Also For instructions on how to customize reason codes and messages, see the FICO Origination Manager Decision Module Administrator's Guide.
