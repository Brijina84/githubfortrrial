Managing Data Methods

Use the options on the list and properties pages for data methods to create, edit, validate, test, copy, and delete data methods.

Covered Topics
  • Creating a Data Method
    Use the Create button on the Data Methods list page to add a new data method.
  • Adding an Input Parameter
    Use the Add icon in the Input Parameters section to define the parameters (input variables) to be passed to a specific data method and used in its expression.
  • Adding a Local Variable
    Use the Add icon in the Local Variables section to define the local variables to be used in a data method expression.
  • Validating an Expression Statement
    When you have constructed a data method expression, validate it to make sure that there are no errors.
  • Testing a Data Method
    After you have created, validated, and saved a data method, use the Test icon on the Data Method Properties page to test the data method.
  • Copying a Data Method
    Use the Save As icon in the Data Method Properties page to add a new data method based on an existing one.
  • Deleting a Data Method
    Use the Delete icon on the Data Methods list page to delete data methods that are no longer needed.
