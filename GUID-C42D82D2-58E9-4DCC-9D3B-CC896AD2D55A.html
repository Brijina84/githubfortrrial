Fees

Use the fee table to define loan fees. These values are used to set and assign the fees for products.

Administrators with the proper permissions can add, modify, and delete entries from the loan fee table.

Covered Topics
  • Adding Entries to the Fee Table
    As an administrator with the proper permissions, use the Fees page to define all the fees you wish to make available in the Application Processing Module system. You then reference the fees you want to apply to the product in the product definition.
  • Modifying the Fee Table
    Use the Fees > Edit <Fee Name> page to modify variable fee entries.
  • Deleting Entries from the Fee Table
    Administrators with the proper permissions can delete variable fee entries.
  • Activating and Deactivating Fees
    Each variable loan fee value in the Application Processing Module has a status of either Active or Inactive.
