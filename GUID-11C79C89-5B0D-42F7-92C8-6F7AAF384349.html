Verification Report

When you verify items, the Decision Module generates a report with the verification status of the item and error details.
When you verify an item from the properties page, the Decision Module displays the verification report in the Verify <item name> dialog box. The verification status can include the following details:
  • Color-coded summary to show the validation status of the item.
  • Item name, item type and error message.
  • Items verified and details

When you run Verify All from the tools tab, the verification status and the details of the items verified appears in the same browser page. You can select Showing errors only or Show all to see the details of the verification report. By default the verification report shows items that fail to verify.

When you verify the configured items in the entire repository or all items of an item type, the Decision Module includes a summary of the verification status and a table with the following details:
  • The item name.
  • The item type.
  • The verification status of the item. Click the verification status to get error details.
