Verifying Synchronous Consumer Credit Applications

When a synchronous consumer credit application contains information that needs to be verified, it is placed in Pending Verification status. Examples of such information are the applicant's monthly income or time with the current employer. The application verification step must be added to the synchronous workflow for processing applications. Application verification is necessary when the system flags an application for verification in the course of processing the application.
  1. To update and resolve an application in Pending Verification status, open the application.
  2. Review the list of items to be verified in the Verify Application pane on the Application page.
    The items vary based on your company's configuration. With the preconfigured Application Processing Module, the following items can be configured to be flagged for verification:
    • Monthly income greater than a specified value for either applicant
    • Time with current employer less than a specified period for either applicant
  3. Verify the relevant details.
    • For items that have been verified, select Verified.
    • For items that have not been verified, select Not verified to decline the application.
  4. For each item, select the verification reason from the drop-down list and enter notes about the verification process.
    Note that verification reasons are available for selection from the drop-down list only if your system administrator has configured those domain values.
    See Also For more information on domain values, see the FICO Origination Manager Application Processing Module Administrator's Guide.
  5. (Optional) To save the verification data, click Save. The system indicates that the application has been partially verified. You can return later to complete the verification process.
    You can also use the Exit Without Saving option.
  6. Click Complete Verification.
    Note that verification of the application can be completed only after you have verified all the items that were flagged.
  7. (Optional) To withdraw an application with unverified items from the system, click Withdraw Application.
