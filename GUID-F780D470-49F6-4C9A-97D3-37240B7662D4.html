Management Tools

Use the Application History feature to monitor the processing steps carried out on an application by viewing application processing history. You can also use the Application Processing Module reporting capabilities if you have the proper permissions (note that only users with system administrative roles can access and manage reports).

Covered Topics
  • Viewing an Application's History
    Use the Application History option to view an application's entire processing history (from the start, through Decision Module evaluation and verification, to the final decision) and the sequence of all user or system activities for an individual application, including any related documentation.
  • Viewing Applications in Error
    Use the Processing Errors option in the task menu to view erroneous applications.
  • Application Processing Module Reports
    The Application Processing Module provides a variety of reports to assist you in evaluating the effectiveness of your system. If the role to manage reports has been assigned, you will have access to view the reports.
