Uploading the Excel Lookup Value Files

Upload the Excel files that contain the lookup values from the Excel Lookup Values option on the Tools tab. You can upload one file at a time.
You must have SMW system administrator privileges to view and use the Excel Lookup Values option.
  1. In the Tools tab, select Excel Lookup Values from the left navigation pane.
  2. In the Excel Lookup Values page, click the Add icon.
  3. Browse to the location of the Excel files, and select the file that you want to upload.
    Note The page lists the amount of occupied space in the repository. Ensure that the files you upload do not exceed the free space available in the repository. If the file you want to upload is larger than the free space available, free up sufficient space by deleting existing files that are currently not being used.
  4. Click Upload.
