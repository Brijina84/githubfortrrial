Color Key for Highlighting Differences

The Visual Differences window highlights the differences between two item versions using a color key.


Comparison Category Description
Added Any items detected in the right pane that are not in the left pane.
Deleted Any items detected in the left pane that are not in the right pane.
Modified Items that are in both panes but have different values.
Order Changed Items that are in both panes, with the subcomponents in a different order. For example, the order of the rules in the Rules table in the Ruleset Properties page.

If you change an item's order as well as its property, the order change takes precedence in the highlighted differences. For example, if you change the order of a row in the Rules table of a ruleset, and also modify the rule's severity, the rule row is displayed in orange.
