Deleting Loan Sources

Use the Loan Sources page to delete loan source information from the system. Administrators with the proper permissions can delete a loan source only when there are no applications associated with that loan source (that is, it is not referenced within an existing application).

When you delete a loan source, you cannot create a new loan source that has the same name as the deleted loan source.

  1. In the administration navigation pane, click Loan Sources.
  2. (Optional) In the search area field, enter search filter criteria and click Search.
  3. Select the loan source to delete and click the Delete Loan Source icon for that row.
  4. Click OK in the confirmation dialog box that appears.
