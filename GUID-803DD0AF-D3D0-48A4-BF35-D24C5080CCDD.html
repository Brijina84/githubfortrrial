Opening an Application

When you are working on an application and want to open a different application, use the Open application number field that appears in the upper right corner of the page.
  1. In the Open application number field, enter the ID number of the application you want to open.
  2. Click Open.
  3. (Optional) Click Next Case to the open the application available after the application you are currently viewing.

    When you click Next Case after you reach the last application in the queue, you will return to the Queues page if you do not have any new applications in that queue.

Note When you are working on the priority queue, the applications are displayed in the sequence that is determined by your administrator.
