Unlocking Applications

Administrators with the proper permissions can release locked applications when write access to a locked application is immediately required and the user cannot be contacted.

When a user opens an application, it is automatically locked so that other users or processes have read-only access to the application and cannot work on it at the same time. The default lock lasts for five minutes. That is, you will receive an unlock prompt every five minutes if you keep the browser open and do not perform any action. The unlock duration is a setting that is configured during installation. Administrators can explicitly unlock applications, and configure the unlock durations. Usually, the default lock duration is set in minutes, for example, between five and 30 minutes. You can extend this timeout period using FICO Application Studio Designer. For more information, see the FICO Origination Manager Application Processing Module Developer's Guide.

  1. In the task menu, click Security and then in the security navigation pane, click Locked Applications.
  2. (Optional) In the search area field, enter search filter criteria and click Search.
  3. Select the applications to unlock and click Unlock.
