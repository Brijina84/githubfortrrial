The Visual Differences Window

The Visual Differences window displays the results of comparing two versions of an item. The window contains two panes, with the earlier version shown in the left pane, and the later version shown in the right pane. The content is in read-only format, with the differences highlighted.

Example

The following screenshot shows an example of the Visual Differences window that is displayed when two versions of a data method are compared.


Visual Differences window for data methods

The Data Method Properties pages for the two versions, with additions, modifications, deletions, and order changes highlighted, are displayed. Note that content that is highlighted in green exists only in the later version, and content that is highlighted in pink exists only in the earlier version. Table rows that are added, deleted, or reordered are highlighted entirely. In table rows that are modified, only the modified portion is highlighted.

Navigation for Additional Differences

The Visual Differences window displays a Jump to drop-down menu if there are additional differences that can be viewed for subcomponents. For example, when you compare two versions of a ruleset where a rule's condition has changed, the name of the changed rule appears as an option in the Jump to menu. You can click the name to navigate to the details of that rule and view the changes.

The Jump to menu is available for the following items:
  • To navigate to rules from a ruleset
  • To navigate to products and product selection trees from a category

The following screenshot shows the Jump to menu in the Visual Differences window for categories. The category has a product called LOC and a product selection tree called psTreeSMB, both of which appear as options in the Jump to menu. Clicking either of these options opens a new Visual Differences window that displays the changes in the details of the corresponding product or product selection tree.
