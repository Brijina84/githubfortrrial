TimestampFormat

Syntax: TimestampFormat(timestamp,formatString)

Description: Converts a timestamp to string format, using a specified format.

Inputs:

  • timestamp: The timestamp being formatted.
  • formatString: The format in which the duration string is to be stored. It can be any format string that is recognized by java.text.SimpleDateFormat, as described in the following documentation: 
    http://java.sun.com/j2se/1.4.2/docs/api/java/text/SimpleDateFormat.html

Return Type: string

Returns:

  • A string that represents a specific timestamp, in a specified format, or null if unable to generate the string.
  • If formatString is null or contains characters that are not valid format “pattern letters”, the function returns an empty or zero-length string and also throws an exception.
  • If timestamp is null, the function returns null.
  • If timestamp is not a valid timestamp value, or if formatString is not a valid string value, the function throws an exception.

Example: 

localString := TimestampFormat(Now(),“yyyy.MM.dd G ‘at’ HH:mm:ss z”);
// Returns 2005.12.16 AD at 11:50:50 PST
