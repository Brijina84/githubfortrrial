Creating and Mapping Score Model Variables

Create and map variables to data methods or fields in the object model. These variables determine the data methods and fields that are evaluated when the score model is executed. After you set up the variables, you can build a score model that uses them as inputs.
For a score model, you can specify up to 30 variables of each of these types:
  • Integer (integer1-integer30)
  • String (string1-string30)
  • Real number (real1-real30)
Before you set up variables, make sure you can edit the score model:
  • If the score model has been checked in, click Check Out.
  • If this is a new score model that has not been checked in yet, click Edit.
  1. In the Manage Variables section of the Score Model Properties page, click the Add icon.
    A variable is added with the following properties and default values:
    Property Default
    Name variable#
    Score model variable name integer#
    Description None
    Object or reference Application
    Field or data method StrategySelectionRandomNumber
    Type integer
    Data object reference None
  2. Click the variable's name to modify the properties.
    The Variable Properties page appears.
  3. (Optional) Modify the name of the variable.
  4. (Optional) Add a description.
  5. Select the data type for the variable from the Type drop-down list.
    You can select integer, string, or real.
  6. Select a corresponding value from the Score model characteristic drop-down list.
  7. Click the Change link for the Variable maps to field to modify the field or data method that the variable maps to in the object model.
    The Object Model Browser appears.
    1. Select a field or data method.
      You can browse or search for a data method or field in the object model.
    2. Click Select.
  8. Specify whether you want to use a data object reference. If you select Yes, select the data object reference from the drop-down list that is displayed.
  9. Click Save.

Repeat this process for all the variables you want to map.
