Service Error Type Values

These values are used by the Data Acquisition Module and Analytic Module adapters.

Service Error Types Set by the Data Acquisition Module Interface
Value Description
DAM_INFO The service call was successful.
DAM_WARNING A warning was returned but the call was successful.
DAM_INQUIRY_ERROR The data acquisition provider processed the request, but returned an abnormal bureau report error with a value of 'E' in <abnormalReportCD>. The submitted subscriber number was invalid, or there was a transmission error between the CRA and the data acquisition provider. Determine the cause and resubmit.
DAM_USER_ERROR A user input error has been encountered. Correct the input and resubmit.
DAM_SYSTEM_ERROR A system error has been encountered. Contact FICO Product Support if you cannot resolve the error.
Service Error Types Set by the Analytic Module Interface
Value Description
AM_INFO The service call was successful.
AM_WARNING A warning was returned but the call was successful.
AM_USER_ERROR A user input error has been encountered. Correct the input and resubmit.
AM_SYSTEM_ERROR A system error has been encountered. Contact FICO Product Support if you cannot resolve the error.
