Maintaining Users

Administrators with the proper permissions can search for users and update user profiles. In addition, you must have the appropriate permissions listed for each task corresponding to the panes on the page.

You can edit users in a group if you are explicitly assigned to manage those users and groups and if you have the appropriate permissions. You can also activate or deactivate a user on this page.

Covered Topics
  • Finding and Viewing User Information
    As an administrator with the proper permissions, you can search for users whose information you want to view or update.
  • Activating and Deactivating Users
    Each user in the Application Processing Module has a status of either Active or Inactive.
  • Changing and Resetting Passwords
    You can reset a password when a user is locked out of the system, has forgotten the password, or if the password expires.
  • Updating User Information
    Administrators can update user information such as the user's name and the roles and groups to which the user belongs on the Users > Edit <User Name> page.
