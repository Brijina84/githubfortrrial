Lists of Values

Many elements in the Application Processing Module domain model can contain enumerated values. Some values may be pre-defined by FICO as domain values as part of the default Origination Manager installation; others are suggested values and may be configured for you to assist your implementation process.

This section lists and describes the enumerated values. It includes information on the FICO Application Studio Designer module and Application Processing Module UI page in which the values can appear.
