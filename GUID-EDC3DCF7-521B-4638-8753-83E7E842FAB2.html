Base Applications

A base application is a standard input application in XML format.
Use the Base Applications tab to create a base application. Alternatively, you can import a standard input application in XML format.
Note The maximum file size allowed for the .xml file is 1MB.

You can create a base application that defines item types and their values such as a category, product or a data object reference. Then add test data to create scenarios to test with a base application.

The following example of a base application defines the product category and product : 
<Application ProcessingRequestType="DM" DeliveryOptionCode="sysid" >
	<CreditRequest ProductCategory="Revolving" ProductCode="SilverCard"/>
</Application>

Once you create a base application, the base application is saved in a private workspace in the SMW application server. You can retrieve a base application to retest. Optionally, you can create a copy of the base application and modify it to test different components and strategies.

Manage base applications from the Base Applications list page. You can perform the following actions from the Base Applications list page:
  • Create a base application
  • Import a standard input application in XML format
  • Search for a base application by name or description
  • Create a copy of a base application
  • Delete a base application
  • Sort base applications
