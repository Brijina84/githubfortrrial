Conditional Statements

Conditional statements are statements that perform different actions depending on the particular inputs they are evaluating. Data methods support two kinds of conditional statements:
  • if-then
  • if-then-else
Covered Topics
  • If-Then Statement
    Use an if-then statement when you want a specific action (or actions) to be performed if a condition is true.
  • If-Then-Else Statement
    An if-then-else statement is an extension of the if-then statement that provides for an action to be performed if a condition is true, and a different action to be performed if the condition is false.
  • Comparison Expressions in Conditional Statements
    You can use simple and complex comparison expressions in conditional statements.
