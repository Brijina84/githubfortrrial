AttributeValue

Syntax: AttributeValue(string1, string2)

Description: Returns the value of the attribute based on the XPath.

The XPath is applied to the application element and must be constructed starting from the application object. Only the first attribute value is returned if multiple attributes have the same name.

Inputs:

  • string1: String containing the filter for the XPath.
  • string2: String containing the name of the attribute.

Input XML: 

<Application DeliveryOptionCode="sysid" ProcessingRequestType="DA">
  <CreditRequest ProductCode="prodTest" ProductCategory="catTest">Example</CreditRequest>
   â€¦

 </Application>

Return Type: string

Returns: The value of the attribute based on the XPath.

When the first argument of AttributeValue is a zero length string, for example, AttributeValue("",""), the following error occurs: Error :com.blazesoft.util.xpath.NdXPathException: Error compiling the XPath expression ''. Empty expression!

If certain arguments are null, for example, AttributeValue(null,null), the following error will occur: Error :java.lang.ClassCastException: com.blazesoft.util.NdSpecialValueException.

You will need to provide the correct input for these arguments.

Example: 

AttributeValue("/Application/CreditRequest","ProductCategory");  // Returns catTest
AttributeValue("/Application/CreditRequest","ProductCode");  // Returns prodTest
