Working with Tools

Covered Topics
  • Strategy Component Snapshot
    A strategy component snapshot is a date-specific report of the items in your business strategies.
  • Updating the Workspace
    To get the item changes that other users of the SMW have checked in, update your private workspace. Updating synchronizes your private workspace with the configuration repository.
  • Manage Object Model
    Use the Manage Object Model option in the Tools tab to view the Object Model schema and its version history.
  • Managing External Data
    When constructing a data method expression, you can insert lookup values. You can import a data file into the Business Strategies workbench and execute a lookup on that data file, especially when the data is too large and when you require versioning capability.
  • Auditing Strategy Changes
    You can audit business strategies by generating a list of changes made to the configured components within a specific period.
  • Publish Rules
    All the business strategies or set of business rules that you author in the business strategies workbench must be compiled and validated. When you publish rules, a decision service is generated and deployed on successful validation. The decision service is available as a REST web service. The decision service can be called at any point from a business process or workflow to decision a credit application.
  • Excel Lookup Values
    When constructing a data method expression, you can insert lookup values from Excel files. Use the Excel Lookup Values option in the Tools tab to upload, download, and delete the Excel files that are used in data methods for decision making. You can upload files with the .xlsx extension only.
  • Custom Files and Functions
    Use the Custom Files and Functions option in the Tools tab to upload the JAR files that contain the custom functions to the repository, download the uploaded files, and delete uploaded files. Custom functions are called from data methods. When constructing a data method expression, you can insert custom functions. In a JAR file, define custom functions based on your business needs. When you upload the file, the functions are available for use in the data methods.
  • Manage Repository
    Use the Manage Repository option in the Tools tab to import and export the configuration repository during a migration process or to save a backup. You can import and export the repository.
