Copying a Product Selection Tree

Use the Save As icon in the Product Selection Tree Properties page to create a product selection tree from an existing product selection tree.
  1. Select the product selection tree you want to copy from the product selection tree list.
    The Product Selection Tree Properties page appears.
  2. Click the Save As icon to copy the product selection tree.
  3. Enter an unique name for the new product selection tree in the Name field.
  4. Click Save As.

    The product selection tree Properties page of the new product selection tree appears.

  5. Optionally, click Edit and modify other properties.
    Note If you delete a product selection tree result, a message appears asking you to confirm deletion.
  6. Click Save.
    Note Under certain circumstances, copying a tree can cause an error to occur. To prevent this problem, carefully check mappings when you copy a tree.
