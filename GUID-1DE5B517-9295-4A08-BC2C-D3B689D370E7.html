Document Template Keywords

Keywords can be used in document templates. The values are taken from the domain model entity attributes with no change in formatting.

Use keywords to insert information about an application into correspondence for a customer.

Covered Topics
  • Creating Document Template Keywords
    Create keywords to use in document templates. Using keywords in a template allows a user to insert information from the application into the document during the document generation process.
  • Modifying Document Template Keywords
    Use the Document Templates > Edit <Keyword Name> page to maintain keywords.
  • Deleting Document Template Keywords
    You can delete document template keywords that are no longer necessary.
  • Activating and Deactivating Document Template Keywords
    Each document template keyword in the Application Processing Module has a status of either Active or Inactive.
