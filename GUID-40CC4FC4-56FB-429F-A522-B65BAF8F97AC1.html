Creating and Mapping Variables

Create and map variables to data fields in the object model. Variables determine the fields that condition nodes evaluate when the tree is run.

You must check out the item type to modify the properties.

  1. In the Manage Variables section of the properties page, click the Add icon.
    A variable appears in the Manage Variables section. You can select the variable and modify the default field values. The following table describes the variable properties and their default values:
    Property Default
    Name variable#
    Description None
    Type Integer
    Variable maps to Application.StrategySelectionRandomNumber
    Use data object reference None
  2. Click the variable name to modify the variable properties.
    The Variable properties page appears.
  3. Optionally, modify the name of the variable.
  4. Optionally, add a description.
  5. Select the data type for the variable from the Type list. You can select Integer, String, or Real from the list.
  6. Select the Change icon to modify the field or data method that the variable maps to in the object model.
    The object model browser appears.
  7. Select values for the Object or Reference field.
  8. Select a value for the Variable maps to field.

    You can browse or search for a field or data method in the object model.

  9. Continue to select fields and data methods for the variables in the tree, and click Save.
