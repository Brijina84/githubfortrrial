Data Methods that Return Values

Within the data method’s expression, you can specify what value will be returned by the method. In addition, you can make assignments to the local variables. The data method expression snippet shown in the following table returns a value for x if x is greater than y; otherwise it returns a value for y.
Conditional Results
Item Value
Expression Expression 
if (x > y) then
result := x
else
result := y
endif

The snippet shown in the following table just returns the value 5.

A Very Simple Expression
