Val

Syntax: Val(theString)

Description: Converts a string to a number.

Inputs:

  • theString: The source string.

Return Type: numeric

Returns:

  • A numeric representation of theString, generated by including all characters that represent numbers in theString, starting at the leftmost character, and proceeding until either the end of the string or a non-numeric character is encountered.
  • If the string represents a number that has more than six digits to the right of the decimal point, the function returns a value that has been “half-even rounded” to the sixth digit. If the seventh digit is a 5, the function rounds the sixth digit to the nearest “even” number.
  • If theString is null, the function returns null and also throws an exception.
  • If theString is invalid the function throws an exception.
  • If theString is an empty string, the function returns 0.0.

Examples: 

localNumeric := Val(“6”); // Returns 6.0
localNumeric := Val(“0”); // Returns 0.0
localNumeric := Val(“abc”); // Returns 0.0
localNumeric := Val(“a1b2c3”); // Returns 0.0
localNumeric := Val(“10,000,000”); // Returns 10.0
localNumeric := Val(“6.05”); // Returns 6.05
localNumeric := Val(“6.0000005”); // Returns 6.000001
localNumeric := Val(“6.0000004”); // Returns 6.000000
localNumeric := Val(“-1”); // Returns -1.0
localNumeric := Val(“9%”); // Returns 9.0
localNumeric := Val(“1a2b3c”); // Returns 1.0
