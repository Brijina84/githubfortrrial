StartsWith

Syntax: StartsWith(string1, string2)

Description: Returns true if this string starts with the specified prefix. This function is case insensitive.

Inputs:

  • string1: String containing the character to be compared against string2.
  • string2: String containing the specified prefix.

Return Type: boolean

Returns: True if the string starts with the specified prefix.

Example: 

StartsWith("ABCD","AB");  // Returns True
StartsWith("ABCD","CD");  // Returns False
