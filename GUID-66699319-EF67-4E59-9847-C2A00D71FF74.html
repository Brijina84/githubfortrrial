Creating a Product

A product is always associated with a category. In the Category Properties page, use the Add icon in the Products section to create products.

When you create a new category, you can add products to the category. Subsequently, you must check out the category and edit it to add products.

  1. In the categories list page, select the category to which you want to add a product.
    The Category Properties page appears.
  2. Click Check Out and click Edit.
    To add a product, you must check out the category and edit the category.
  3. In the Products section, click the Add (+) icon.
    The Decision Module adds a product with a default name and code.
  4. Click the new product.
    The Product Properties page appears.
  5. In the Name field, modify the name for the product.

    Product name is case-sensitive and must be unique within a category.

  6. Optionally, modify the product code in the Code field.

    Product code must be unique within a category.

  7. Optionally, enter text describing the product in the Description field.
  8. Optionally, select a default decision flow in the Default Decision Flow field.
    Note If you do not define a default decision flow and the product strategy does not select a decision flow, a decision flow will not execute.
  9. Optionally, select a product strategy that references at least one decision flow in the Product Strategy field.
  10. Optionally, enter product active dates in the from and to date fields.
  11. Optionally, you can activate product severity threshold. Select Yes in Use product threshold field. Default is No.
    The Product severity threshold field appears.
  12. If you activate product severity threshold processing in the preceding step, enter an integer greater than 0 in the Product severity threshold field.
  13. If you have defined other products in the current category or have defined other categories, you can assign an up-sell or down-sell product, or a cross-sell category. Select the following fields:
    • Up-sell product
    • Cross-sell category
    • Down-sell product
    Note You can also navigate or preview the product or category.
  14. Click Save and Check In .
    Note
    The following rules apply when you check in products:
    • If you create a category and add products, the products are checked in when you check in the category. Once you check in a category, you can independantly manage the products from the Product Properties page.
    • If you create a category and add products, the category is checked in when you check in a product
    • If you add a product to an existing category, then you can navigate to the Product Properties page to check in the product. You must check out and edit a category to add products.
