Deleting a Score Model Row

Use the Remove current score condition icon in the Score Model Information section to remove a score model row in a scoring scenario.
  1. In the Scenarios list page, select the scoring scenario you want to modify.
  2. If the scoring scenario is not checked out, click Check Out.
  3. If the scoring scenario is checked out, click Edit.
  4. Click the Remove current score condition icon for the score model row you want to delete.
  5. When prompted, click Yes to confirm the deletion.
  6. Click Save.
