Verifying All Items

Use the Verify All icon in the Tools tab to verify all items in the repository or all items of an item type.
  1. In the Tools tab, select Verify All.
  2. Select the items in the entire repository or all items of an item type from the Select drop-down list.
  3. Click Verify.
    The Decision Module displays a summary of the verification status and a detailed verification report. By default only the items that fail to verify appear in the verification report.
The summary of the verification status is in the following format: Verification is complete for the items and the dependent items defined for the item type decision flows. 3 items verify successfully. 2 items fail to verify. See the verification report for more information.

The details of the verification report includes the item name, item type and the verification status. Click the verification status of an item to see errors. To resolve errors, see Verification Error Scenarios
