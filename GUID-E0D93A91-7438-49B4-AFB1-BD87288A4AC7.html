Entering Data for a Synchronous Consumer and Consumer Credit Application

Note If your company supports more than one line of business, be sure to have selected the OM APM Consumer option from the drop-down list in the top-level menu.
For the Consumer LOB, the Create Application page consists of the following panes—Application Details, Personal Details, Employment Details, Financial Details, and Assets and Liabilities.For the Synchronous Consumer LOB, the Create Application page consists of the following panes—Application Details, Personal Details, Employment Details, Financial Details, Assets and Liabilities. If the configuration permits addition of authorized users, then the Authorized Users pane is present.
To enter data in the Create Application page, perform the following steps:
  1. In the Application Details pane, enter the details about the application.
  2. Enter applicant information in the Personal Details pane.
  3. Enter employment information in the Employment Details pane.
  4. Enter financial information in the Financial Details pane.
  5. Enter information about the assets and liabilities in the Assets and Liabilities pane.
  6. (Optional) Click Add applicant.
    Additional application fields appear on the page allowing you to enter similar application details for a co-applicant.
  7. Do one of the following:
    • Click Submit to save the information and continue with the workflow.
      Note The application will not be submitted until all the required fields are filled.
    • Click Save if you want to save the information at this moment, and submit the application at a later time.
    • Click Exit without Saving if you do not wish to save or submit the application.
