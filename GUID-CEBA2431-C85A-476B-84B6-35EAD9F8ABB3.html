Exporting Reports to Excel

If you want to save, further manipulate, or print report information, export the report to Microsoft Excel.
Note
  • The number of records that can appear in an Excel spreadsheet is 1,048,576, so any number of rows greater than that number will not be saved.
  • FICO supports exporting your data to Excel, but does not provide customer support for the Microsoft Excel application.
  1. Display the report with the information you want to export.
  2. Click Export to Excel.
  3. Click Open to view the file or click Save to save the file without viewing it first.
  4. After opening or saving the file, you can work with your information in Excel or print the report if needed.
