Configuring Domain Values

Administrators with the proper permissions can create, modify, delete, activate, and deactivate the available domain values. A "domain value" is a set of information about possible values that can appear in a UI field or XML transaction payload element. The information includes the actual value, the name of the value, and a description of the value, among other items. In the Origination Manager domain model, various entities use domain values that represent lookup values.

Administrators can only edit domain value translations for locales that are enabled within the Application Processing Module. The values you add or edit on a Domain Values page are made available to the user as values in drop-down lists.

See Also For more information on domain values, see the FICO Origination Manager Application Processing Module Developer's Guide.
Covered Topics
  • Creating Domain Values
    Administrators with proper permissions can add new values to existing domain values.
  • Modifying Domain Values
    Administrators can modify or set the status of domain values.
  • Deleting Domain Values
    Administrators with the proper permissions can delete domain value information from the system.
  • Activating and Deactivating Domain Values
    Each domain value in the Application Processing Module has a status of either Active or Inactive.
