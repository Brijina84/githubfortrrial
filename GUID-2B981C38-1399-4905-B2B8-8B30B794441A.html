Adding Reason Codes to a Score Model Reason List

Create one or more reason codes for the reason code list.
  1. Locate the reason code list in the Score Model Reasons list page, and then click the list name.
  2. Click the Edit icon.
  3. Click the Add (+) icon.
    A reason code with a default name appears.
  4. Click the name to open the Score Model Reason Properties page.
  5. Change the default name to a suitable name.
  6. Enter the code.
  7. In the Message field, enter the text that you want to appear when this reason code is returned.
  8. Click the Save icon.
  9. Repeat Steps 3 - 8 to create additional reason codes.
    The rank number is determined by the order in which the reason code appears in the Reason codes table. You can change the order by using the Move Up and Move Down commands on the Reason codes table.
