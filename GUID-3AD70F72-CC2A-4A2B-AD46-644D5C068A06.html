State Values

These values can appear in the following places:
  • The State.Code attribute in the om_domainvalue module in Designer.
  • The BaseContactAddress.StateId attribute in the om_apm_core module in Designer.
  • The address.state, employerAddress.state, and mailingAddress.state attributes in the om_apm_consumer module in Designer.
  • The Domain Values > State page in the Application Processing Module user interface.
