Configuring Score Ranges

Use the Reports Configuration > Score Range Configuration page to define the lower and upper limit score range values for a model category (line of business) used in the Final Score and Approval Rate Decomposition reports.
  1. In the administration navigation pane, click Reports Configuration > Score Range Configuration.
  2. In the Lower Limit field, enter the lower limit score range value for the model category.
  3. In the Upper Limit field, enter the upper limit score range value for the model category.
  4. Select the model category to which you want to apply the lower and upper limits you just set from the Model Category drop-down list.
    You can only set one lower and upper limit score range value for each model category. If you set a lower and upper limit score range value for an existing model category, the updated information replaces the existing information in the table on the Reports Configuration > Score Range Configuration page.
  5. Click Save.
