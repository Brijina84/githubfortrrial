Editing Group Details

Administrators with the proper permissions can view or edit group details and select an Active or Inactive status for the group.
  1. In the task menu, click Security and then in the security navigation pane, click Groups.
  2. (Optional) In the search area field, enter search filter criteria and click Search.
  3. Select the group to modify and click the Edit group icon for that row.
  4. Make the appropriate changes to the group's information. To change a group's status, use the Active or Inactive radio buttons to enable or disable the group's access to the application.
  5. Click Save.
