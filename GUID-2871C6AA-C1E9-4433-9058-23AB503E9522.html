Setting Score Cutoffs

Setting score cutoffs at the population level provides more granularity for decisions made during application processing. The populations come preconfigured with default cutoff values, which you can modify as needed, but you can also add more cutoffs and add populations for other scores, with other cutoffs.
Note The default cutoff values that come preconfigured with the populations may not be appropriate for your business needs. You must determine the cutoff values to use.
  1. In the administration navigation pane, click Populations.
  2. Click Create.
  3. Select a model category for the population. You can only map one population to each model category.
    Note The score cutoffs for different product categories can be different for consumer and small business products and product categories; therefore be sure to select the correct model category.
  4. Enter the description and score cutoff value for the population that you are adding.
  5. Click Create.
