Detach

Syntax: Detach(parentObject, childObject)

Description: Detaches an object from another.

Inputs:

  • parentObject: The object from which the child will be detached.
  • childObject: The object being detached.

Return Type: void

Returns: N/A

Example: 

Detach(Application,personalApplicant);
