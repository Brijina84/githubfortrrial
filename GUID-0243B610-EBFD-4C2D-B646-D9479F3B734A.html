Attach

Syntax: Attach(parentObject, childObject)

Description: Attaches an object to another.

Inputs:

  • parentObject: The object to which a new child object will be attached.
  • childObject: The object being attached.

Return Type: void

Returns: N/A. If childObject is null, or if childObject is not a valid immediate descendant of parentObject, the function throws an exception. Other error conditions are flagged as validation errors when you try to validate or save the method.

Example: 

localApplicant := new personalApplicant;
Attach(Application,localApplicant);
localApplicant.ApplicantCrossReferenceId := “abc”;
