Modifying Loan Sources

Administrators can modify, activate, or deactivate loan source information.

  1. In the administration navigation pane, click Loan Sources.
  2. (Optional) In the search area field, enter search filter criteria and click Search.
  3. Select the loan source to modify and click the Edit Loan Source icon for that row.
  4. Make the desired modifications to the loan source information.
  5. (Optional) To set this loan source as the default loan source, select the Set as default check box.
  6. (Optional) To set the status of the loan source, perform one of the following actions:
    • To set the status to Active, select Active.
    • To set the status to Inactive, select Inactive.
  7. Click Save.
