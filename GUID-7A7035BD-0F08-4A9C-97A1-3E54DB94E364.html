Inserting an Operation

When constructing a data method expression, you can view and select simple operators such as + and -, keywords such as "new", and code templates for "if"conditions and comments.
Tip To insert an operator, press Ctrl + Space at the start of a new line or after a space, then double-click the operator in the displayed drop-down list.

To add an operator using the toolbar option, follow these steps:

  1. Click the Insert an operation icon in the Expression Editor toolbar.
  2. In the Operators panel, click an operator, and close the panel.
The operator is inserted at the cursor location in the editor. If you selected a conditional operator such as if-then or if-then-else, the complete syntax is inserted as shown below. You can then modify the syntax. 
if (true) then true; endif
if (true) then
	true;		 
else
	false;	
endif
