ExcelCellValueLastError

Syntax: ExcelCellValueLastError(dummyArg)

Description: Stores the last error message, if there are errors when an Excel file lookup is performed. Used with the ExcelCellValue and ExcelCellValueIsError functions.

Inputs:

  • dummyArg: The Blaze Advisor framework for custom functions requires an argument. In this case, the dummy argument is empty.

Return Type: string

Returns: The last error message.

Example: For an example, see ExcelCellValue.
