Modifying Domain Values

Administrators can modify or set the status of domain values. When modifying domain values, translations will only display for active locales.
  1. In the administration navigation pane, click Domain Values and then click the Edit icon in the row for which you want to modify the domain value.
  2. (Optional) In the search area field, enter search filter criteria and click Search.
  3. Select the domain value to modify and click the Edit <Domain Value Name> icon for that row.
    Note If you are modifying the Prefix Name, include or exclude the dot after the prefix name, as required. For example, Dr., Mr., Miss, and so on.
  4. Make the desired modifications to the domain value information.
  5. (Optional) To set this domain value as the default domain value, select the Is default check box.
  6. (Optional) To set the status of the domain value, perform one of the following actions:
    • To set the status to Active, select Active.
    • To set the status to Inactive, select Inactive.
  7. Click Save.
