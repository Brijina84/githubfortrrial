Comparison Results for Rulesets and Rules

When you compare two versions of a ruleset, the differences in the properties of the ruleset and its rules are highlighted.
In the the Visual Differences window for rulesets, changes in the ruleset's properties and the Rules table are highlighted using the standard color keys. The Jump to menu is available when there are changes in rule properties or contents.
Note In the Rules table, a rule is highlighted as having been modified if any of its properties has changed. If the only changes are in the rule's contents—for example, a changed condition value in its definition—the rule is not highlighted. However, its name is displayed as an option in the Jump to menu.

To view the differences in a rule's contents, click the rule's name in the Jump to menu. In the new Visual Differences window that appears, differences in the rule's properties, outcome, and definition are highlighted.

To return to the window for ruleset differences, click Ruleset Properties in the Jump to menu.
