Introduction

The FICO® Origination Manager Decision Module decisions credit applications using decision strategies that business users set up and maintain using the Decision Module’s Strategy Management Workstation.
Covered Topics
  • Origination Manager Overview
    FICO® Origination Manager is a multi-module solution with preconfigured components that you can use to create an end-to-end application processing solution based on your company's requirements.
  • Application Processing Module Overview
    The FICO® Origination Manager Application Processing Module is built on the FICO® Application Studio.
  • Decision Module Overview
    The FICO® Origination Manager Decision Module is an account origination product that leverages the full power of the FICO® Blaze Advisor® business rules management system. It provides companies with proven rules-management technology that supports the entire process for designing, building, and maintaining rules-based decision applications.
