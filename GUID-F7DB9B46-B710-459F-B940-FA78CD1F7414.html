Creating a Pricing Scenario

Use the Create button on the Scenarios list page to add a new pricing scenario.
  1. In the navigation pane, click Scenarios.
  2. On the Scenarios list page, click the Create button.
  3. In the Create dialog box, enter a unique name.
    Names must be text strings and can contain alphabets, numbers, and underscores.
  4. In the Description field, enter text describing the decision scenario.
  5. In the Type drop-down list, select Pricing Scenario.
  6. Click Create.
    The new pricing scenario is listed on the Scenarios list page.

    To add a product offering and specify its parameters, you need to edit the pricing scenario. At this point, you can proceed with the following steps to add product offerings, or you can check in the pricing scenario to commit it to the configuration repository. If you check in the scenario, you need to check it out when you are ready to add product offerings.

  7. Click the pricing scenario name on the Scenarios list page.
    The Pricing Scenario Properties page appears.
  8. Click the Edit icon.
  9. For each product offering you want to add, perform the following steps in the Manage Product Offerings section.
    1. Click the Add a new offer icon.
      A row for specifying a product offer is displayed.
    2. In the Offer ID field, enter a text value for representing the offer ID.
    3. In the Rate % field, enter a real number specifying the interest rate.
    4. In the Amount field, enter an integer specifying the amount of the offer.
    5. In the Term (months) field, enter an integer specifying the term of the offer, in months.
  10. Click Save.
