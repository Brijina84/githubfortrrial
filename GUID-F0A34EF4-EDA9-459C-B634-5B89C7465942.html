Reason Codes

You can create three types of score models using the Strategy Management Workstation.
  • Score models that do not return reason codes
  • Score models that return reason codes in rank order
  • Score models that return reason codes ranked by distance from the characteristic baseline score
You specify the score model type when you create a score model.
Note Once the score model is created, you will not be able to change the type.
Covered Topics
  • Score Models That Return No Reasons
    Choose this option when you do not want reason codes returned.
  • Score Models That Return Ranked Reasons
    Choose this option when you want to return reason codes based on their priority order in the reason code list, which is a preconfigured set of 100 reason codes, ranked from 00 to 99.
  • Score Models That Return Distanced Reasons
    Choose this option when you want to return reason codes ranked according to the distance of the generated characteristic score from the characteristic baseline.
