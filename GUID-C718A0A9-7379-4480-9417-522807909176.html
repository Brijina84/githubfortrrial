Deleting Entries from the Fee Table

Administrators with the proper permissions can delete variable fee entries. You cannot delete a fee if it is used within a product.
  1. In the administration navigation pane, click Fees.
  2. (Optional) In the search area field, enter search filter criteria and click Search.
  3. Select the fee entry to delete and click the Delete Fee icon for that row.
  4. Click OK in the confirmation dialog box that appears.
