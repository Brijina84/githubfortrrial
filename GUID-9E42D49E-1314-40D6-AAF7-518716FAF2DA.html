Adding Product Categories

Use the Categories > Create Category page to add product categories to the system. A loan type can only have one product category.
  1. In the administration navigation pane, click Categories.
  2. Click Create.
  3. Enter a code and name for the new product category that you are adding. You cannot create a product category with the same name as an existing one.
    The product category can specify the report purchase and scoring policies, or the various timeout parameters. For example, you could enter a category name such as "Consumer Card" or "Corporate Card" if you select Consumer from the Type drop-down list. Or, you could enter a category name such as "Line of Credit" if you select Small Business from the Type drop-down list.
  4. Select a product category type from the Type drop-down list and optionally, add a description.
  5. In the Business Policy Settings area, move the slider buttons for Purchase bureau report and the Purchase SBSS report to specify whether the bureau report and the SBSS scores are purchased or not.
  6. In the Policy Settings area, specify the following settings:
    Policy Settings 
    Setting 
    Description
    Purchase CRA report CRA report is to be purchased for primary applicant or all applicants.
    Score Scoring policy is to be applied too the primary applicant or to all applicants.
    Evaluation Policy TBD
  7. Click Create.
    Note If you add a new product category, you need to modify the Business Strategies to contain rules for the product category or you will encounter an error.
