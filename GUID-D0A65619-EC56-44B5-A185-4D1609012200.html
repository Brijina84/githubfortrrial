Deleting Items

Use the Delete icon on the list page to delete items that are no longer needed.
  1. On the item's list page, select the item you want to delete.
    If you want to delete a data method, for example, select it form the Data Methods list page.
  2. If the item is not checked out, click Check Out.
  3. Click the Delete icon.
    Note The Decision Module prevents you from deleting an item that is referenced by another item. Use the Where Used search feature to find the references. Before you delete the item, remove all references to it.
  4. When prompted, click Yes to confirm.
    If this is a new item and has not been versioned yet, it is no longer displayed. If the item has been versioned, its name is displayed in strikethrough format; the item is deleted from your private workspace, but still exists in the configuration repository.
    Note It is possible to undo the deletion from your private workspace by clicking Cancel Check Out.
  5. To remove the item from the configuration repository, click Check In.
