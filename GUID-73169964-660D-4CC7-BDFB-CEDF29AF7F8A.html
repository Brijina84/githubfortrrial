Application Processing - User tasks

Learn how to create applications, enter and update application data, obtain credit report and scoring data, and process applications through the system. Data entry clerks, credit analysts, customer service representatives, supervisors, and managers would perform these tasks.

Covered Topics
  • Application Entry
    When you log on to the Application Processing Module, access to the web pages used in the creation, modification, and submission of applications is made available to you based on the roles and groups to which you have been assigned.
  • Application Processing
    Use the My Queues pane on the Home page to select and process applications within queues that require manual intervention, such as review, verification, duplicate check resolution, or credit offer entry. For synchronous consumer applications, the processing happens automatically and you can take the required action on an application.
  • Management Tools
    Use the Application History feature to monitor the processing steps carried out on an application by viewing application processing history. You can also use the Application Processing Module reporting capabilities if you have the proper permissions (note that only users with system administrative roles can access and manage reports).
