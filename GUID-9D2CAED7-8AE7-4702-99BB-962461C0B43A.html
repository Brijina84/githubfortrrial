Managing Decision Flows

The SMW provides a UI-based tool that allows easy creation and manipulation of decision flows. Access Decision Flows from the Item Types tab.

When working with decision flows, it is important to understand how flow branching occurs. Step results are defined in rulesets and decision trees. Step results are used when creating a decision flow step transition to determine which step is to be taken next.

The Result Value column contains result values that have been defined and are used in the decision flow step. The Next Step column contains the name of the step that is taken if the associated result value is returned when this step is executed. The Otherwise result type defines the path taken if any of the following occur:
  • No result is returned
  • The result is an invalid value
  • There is no transition defined for the result

Decision trees and rulesets can provide multiple result values. The Otherwise result type is the only option available in decision table, data method sequence, and score model implementations.

Covered Topics
  • Creating a Decision Flow
    Use the Create button in the Decision Flows list page to create a decision flow.
  • Adding a Decision Flow Step
    Use the Add icon in the Decision Flow Properties page to add steps to a decision flow. The Add icon appears in the editor.
  • Creating Step Transitions Using the Transition Information Section
    Use the Transition Information section in Step Details dialog box to create step transitions.
  • Modifying a Step Transition Using the Step Details Dialog Box
    Use the Step Details dialog box to modify a step transition.
  • Deleting a Step Transition
    Edit a decision flow step or click anywhere in the transition arrow to delete transition between steps.
  • Renaming a Step
    Use the Step Details dialog box to rename a step.
  • Moving a Step to a New Position
    Use the mouse to move a step to a new position in the decision flow and save the position
  • Deleting a Step from a Decision Flow
    Use the Delete icon in the decision flow step to delete a step that is not required in the decision flow.
  • Copying a Decision Flow
    Use the Save As icon in the Decision Flow Properties page to create a new decision flow from an existing decision flow.
  • Deleting a Decision Flow
    Use the Delete icon in the Decision Flows list page to delete a decision flow that is not required.
