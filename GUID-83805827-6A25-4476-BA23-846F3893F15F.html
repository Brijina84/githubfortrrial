Data Method Literal Values

The following table lists the data types for literal values (explicit values such as numbers and text strings), describes the data that is valid for each type, and cites the initial value assigned to local variables of each type if no value is otherwise assigned. Refer to these values when entering or selecting an initial value for a data type.
Literal Values
Data Type Valid Value Initial Value
boolean true, false, null false
date, time, timestamp, duration duration(6,"days"), yyyy-MM-dd HH:mm:ss:S, now() null
numeric null or an optionally signed integer or real number, such as -1234.56, 25, 0, or 0.0 0.0
string null or a sequence of characters or a zero length string
Note For the string data type, do not include quotes when specifying an initial value.
object parent and child objects, for example, Application and personalApplicant null
