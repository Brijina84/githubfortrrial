Creating Applications

There are different options to use when beginning the process of creating an application. Accurate entry of application data is imperative for the Application Processing Module to process applications.
Note Select the appropriate line-of-business template from the drop-down list in the top-level menu if you have been assigned to more than one line-of-business template.
  • On the Home page, click Create Application.
  • From the task menu, click Create Application.
