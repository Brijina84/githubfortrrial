Inserting an Excel Lookup Value File in a Function

When constructing a data method expression, you can insert functions. When inserting functions, you can use the data from an Excel file for the function.
  1. Click the Insert a function icon icon in the Expression Editor toolbar.
    In the Functions panel, the Available functions drop-drown list displays all the predefined and custom functions that are available. For each function, the name and type of arguments, and the return type are displayed.
  2. Perform one of the following actions:
    • Double-click the function in the drop-down list.
    • Click the function in the drop-down list, then click Insert.
  3. For the Excel file input parameter, perform the following steps:
    1. Move the cursor to the Excel file input parameter parameter.
    2. Click the Insert Excel file lookup icon.
    3. Select the Excel file from the list of Excel files that you have uploaded.
    4. Click Insert.
  4. Insert the remaining inputs for the function.
  5. Close the Functions panel.
  6. Click the Save icon.
