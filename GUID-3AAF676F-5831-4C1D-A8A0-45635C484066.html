Experian CRA Score Reason Code Values

The values listed in the Experian CRA Scoring Reason Code Values table only apply to FICO® scores that have a CRA Scoring Model Response value of F1, F2, F3, F4, F6, or FB. The values listed in the Experian CRA Scoring Reason Code Values (NextGen Scores) table only apply to FICO® scores that have an online credit bureau scoring model code of F9. For score factors that apply to other scores, refer to the manual from Experian.

The FICO Scoring Value appears in the CRAScoreReason.Code attribute in the om_apm_cra_core module in Designer.
