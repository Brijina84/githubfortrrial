Verifying an Item

Use the Verify icon in the properties page of any item to verify the item.
  1. In the Item Types tab, select an item type to view the list page.
    The item type list page appears with a list of items.
  2. Click an item to view its properties page.
    The properties page appears with a tool bar that contains the Verify icon.
  3. Click the Verify icon.
    Note After you start the verification process, it cannot be interrupted.
    Note If the item contains more than one invalid item, the verification process stops on detecting the first error. The error message indicates the item type that contains an error. Because the verification process stops at the first error, you must fix the error and continue the verification process until all errors are detected and resolved.
    When the verification is complete, a report appears in the Verify dialog box.
  4. If there are verification errors for an item, click the item to navigate to the item properties page.
    Resolve verification errors. For more information, see Verification Error Scenarios.
