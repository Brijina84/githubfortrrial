Top-level Menu

The top-level menu provides links to frequently used features you might want to access while you are working and also allows you to navigate between different lines of business. It also displays the first and last name of the logged on user. For example, in the following figure, John Smith is the name of the user currently logged on to the system.
Note Access to multiple lines of business, and the options available based on the solution that is built for your company, and depend on the roles that you have been assigned.

Use the top-level menu to access the following links from the Home page or anywhere within the Application Processing Module system:
  • Line of Business drop-down list—Allows you to switch between the line-of-business templates available to your company. When switching between line-of-business templates, there will be differences in the fields for queues, search, processing errors, duplicate check, and other line-of-business specifics on some configuration pages. Details about these differences are discussed later in specific topics in this guide.
  • Home—Displays the Home page from any point within the system.
  • Favorites—Displays a list of applications that have been marked as favorites. Use the Add to Favorites option on the Application page to add links to this list.
  • About—Displays Application Processing Module version and copyright information.
  • Help—Displays the Application Processing Module Online Help.
  • Log off—Logs you off of the system. If you are editing an application or configuration component, the system prompts you for your confirmation before logging you off.
