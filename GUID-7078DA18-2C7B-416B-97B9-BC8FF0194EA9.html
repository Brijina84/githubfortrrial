Viewing Application Overview Data for a Small Business Credit Application

The Application Overview pane on the Application page for a small business credit application provides the following summary information about the principal:
  • Name
  • Present address
  • Phone number
  • Registration number
  • Number of years in business
  • Number of employees
  • SBSS Score (If the SBSS Score is less than the population score cut-off value, the value in the score bar appears red. If the Credit Score is equal to or greater than the population score cut-off value, the score bar appears green.)
  • Loan to Value Ratio (provided as a percentage)
  • Data for the assets and liabilities, as well as the income and expense for three years are displayed in the form of line graphs.

The Application Overview pane contains a Principal Overview sub-pane that contains the following current details for the principal and, if there is more than one principal, for the additional principals.

  • Name
  • Home address
  • Phone number
  • Identification type and number
  • Number of years in business
  • Percentage of ownership of the business
Note If you choose to mask certain personal identifiable information (PII) on the UI, you must use the field masking or component authorization techniques available for these pages. For more information, contact FICO Product Support.
