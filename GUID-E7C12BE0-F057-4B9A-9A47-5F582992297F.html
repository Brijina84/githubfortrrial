Application Entry

When you log on to the Application Processing Module, access to the web pages used in the creation, modification, and submission of applications is made available to you based on the roles and groups to which you have been assigned. These roles and groups control what you see on a page and the functions you can perform. Contact your security administrator, if necessary, to obtain additional permissions.
Covered Topics
  • Creating and Submitting Applications
    To create and submit an application, follow all data entry guidelines, verify that all required data is present, and submit the application or save it for additional data entry at a later time.
  • Editing an Application
    If you have the appropriate permissions, you can edit applications that have been saved or submitted, but have not been processed to completion.
  • Withdrawing Applications
    When an application has to be withdrawn (for example, when a credit applicant elects to do so), you can locate the application and withdraw it from the system.
