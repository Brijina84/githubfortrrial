InRange

Syntax: InRange (value,min_value, max_value)

Description: Checks whether the value is within a specified range.

Inputs:
  • value—Numeric, value to be checked
  • min_value—Numeric, minimum value of the specific range
  • max_value—Numeric, maximum value of the specific range

Return type: Numeric

Returns:
  • value—If the value falls between min_value and max_value.
  • min_value—If the value < min_value.
  • max _value—If the value > max_value.
Example:
  • InRange(10,20,30) ---- Returns 20
  • InRange(20,10,30) --- Returns 20
  • InRange(30,10,20) --- Returns 20
