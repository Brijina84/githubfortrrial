Copying a Decision Flow

Use the Save As icon in the Decision Flow Properties page to create a new decision flow from an existing decision flow.
  1. In the Decision Flow Properties page, click the Save As icon.
  2. Enter an unique name for the new decision flow in the Name field.
  3. Click Save As.

    The Decision Flow Properties page of the new decision flow appears.

  4. Click Edit to modify the description and any other properties you want to change.
  5. Click Save.
