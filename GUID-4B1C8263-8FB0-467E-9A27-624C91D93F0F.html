FormatNumber

Syntax: FormatNumber(number)

Description: Returns a number based on the format passed in. For more information on formatting, see https://docs.oracle.com/javase/7/docs/api/java/text/DecimalFormat.html.

FormatNumber can handle null, infinity (both positive and negative), and NaN (Not a Number) arguments.

Inputs:

  • number: A number
  • string: The format of the string to apply to the number

Return Type: numeric

Returns: A number based on the format passed in. Includes special handling of number as follows:
  • If number is a number, the function returns the original value.
  • If number is NaN, the function returns NaN.
  • If number is larger than the largest supported numeric value, the function returns infinity.
  • If number is smaller than the smallest supported numeric value, the function returns negative infinity.

Example: 

FormatNumber(null,null));  // Returns NaN
FormatNumber(6/0,"0.0000000000"));  // Returns Infinity
FormatNumber(-6/0,"0.0000000000"));  // Returns -Infinity
FormatNumber(SQRT(-6),"0.0000000000"));  // Returns NaN
FormatNumber(0/0,"0.0000000000"));  // Returns NaN
FormatNumber(0.12345,null));  // Returns 0.12345
FormatNumber(0.1234511111111,"0.0000000000");  // Returns 0.1234511111
FormatNumber(0.123452222222,"#");  // Returns 0.0
