System Administration

Covered Topics
  • Score Cutoffs and Populations
    A score cutoff is a score value used when making decisions that an administrator with the proper permissions designates. Decisions can be based on whether scores fall below, are equal to, or exceed the cutoff value.
  • Configuring Domain Values
    Administrators with the proper permissions can create, modify, delete, activate, and deactivate the available domain values. A "domain value" is a set of information about possible values that can appear in a UI field or XML transaction payload element. The information includes the actual value, the name of the value, and a description of the value, among other items.
  • Document Templates
    Document templates are used to define the layout for letters and other documents that are generated within the Application Processing Module.
  • Organizational Structure
    Use the Organizations page to display the client organization hierarchy, create a new client organization, and view and edit client organization information.
  • Reports
    The Application Processing Module provides users with five standard reports.
  • Customizing Search Fields
    Data architects can customize the list of search filter fields by using FICO Application Studio Designer to extend entities in the om_apm_consumer, and the om_apm_sb modules for the Consumer and the Small Business templates respectively.
  • Managing Translations
    Administrators with the proper permissions can download and upload translation files for the display of runtime resources used in creating queues and menus.
