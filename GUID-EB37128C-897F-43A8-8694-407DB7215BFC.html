CountStr

Syntax: CountStr(string1, string2)

Description: Calculates the number of non-overlapping occurrences of string1 in string2. This function is case-sensitive.

Inputs:

  • string1: The string to count in string2
  • string2: The string in which to count non-overlapping occurrences of string1.

Return Type: numeric

Returns:

  • A number representing the total number of non-overlapping occurrences of a string.
  • If string1 or string2 is null, the function returns null.
  • If string1 or string2 is not a valid string value, the function throws an exception.

Example: 

localString := CountStr(“1”,”1222232111214555XX”);  // Returns 5.0
