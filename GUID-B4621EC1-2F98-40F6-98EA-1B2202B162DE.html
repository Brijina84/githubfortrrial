ReplaceString

Syntax: ReplaceString(string1, string2, string3)

Description: Replaces the specified string.

Inputs:

  • string1: String containing the original information.
  • string2: String containing the regular expression (character to specify and recognize string1). For more information on regular expression usage, see http://java.sun.com/developer/technicalArticles/releases/1.4regex/.
  • string3: String to replace string1.

Return Type: string

Returns: The specified string.

If certain arguments are null, for example,
  • result := ReplaceString("AB#D",null,"C");
  • result := ReplaceString("AB#D","#",null);
  • result := ReplaceString("AB#D",null,null);

the following error will occur: Error :Argument conversion failure for argument number <argumentNumber> Cannot convert an Advisor exception value to a Java value of type 'java.lang.String'. Value exception type = unknown.

You will need to handle the null values being passed. Note that other combinations of null arguments will return "unknown".

Example: 

ReplaceString ("AB#D","#","C");  // Returns ABCD
ReplaceString ("ABC#","#","D");  // Returns ABCD
