Locating Where Used Items

Use the Where Used option on an item's properties page to find where the item is being referenced.
  1. In the left navigation pane of the Item Types tab, click the item type for the item whose references you want to determine.
  2. On the list page for the item type, click the name of the item.
  3. On the item's properties page, click the Where Used icon.

    When the search is complete, the results are displayed in a separate Where Used dialog box. The Where Used dialog box lists the referencing items, and provides links to those items.

  4. Perform one of the following actions in the Where Used dialog box:
    • To return to the properties page of the item, click Close.
    • To view the details of a referencing item, click its name in the Where Used dialog box. The referencing item's properties page is displayed.

      For example, if you have a data method dm_Util_MonthlyAmount which is referenced by another data method dm_App_WeightedMonthlyIncome, the Where Used window will display a link called dm_App_WeightedMonthlyIncome. Clicking this link will take you to the corresponding Data Method Properties page.
