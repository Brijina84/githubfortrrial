The Home Page

After you log on, the Home page appears; it provides access to your applications, queues, reports, system administrative options, and items you have recently worked on within the system. If you have access to multiple line of business templates, you can also switch between the templates (for example, Consumer, Synchronous Consumer, and Small Business) from the Home page. The items that are available to you depend on the roles that you have been assigned.

Covered Topics
  • Create Application
    The Create Application option on the Home page provides a way to access the Create Application page and create applications for new or existing customers. Based on whether you have selected OM APM Consumer, OM APM Synchronous Consumer, or OM APM Small Business from the top-level menu, you can create a consumer, synchronous consumer, or small business application, respectively.
  • Search Application
    The Search Application option on the Home page provides a way to access the Search page and search for existing applications in the system.
  • System Admin
    The System Admin option on the Home page provides a way to access the administration navigation pane and perform system administrative functions such as creating synchronous workflows, maintaining queues, categories, products, CRAs, business bureaus, and organizations, if you have been granted administrative privileges. If your role does not permit you to access the administration navigation pane, the System Admin option is hidden from view.
  • Security
    The Security option on the Home page provides a way to access the security navigation pane and perform functions such as unlocking applications and managing clients, users, and groups (if you have been granted security administrative privileges). If your role does not permit you to access the security navigation pane, the Security option is hidden from view.
  • My Queues
    The My Queues pane on the Home page provides a way to view and access queues you are assigned to.
  • Recently Worked On
    The Recently Worked On pane on the Home page provides a way to access recently viewed items so that you can quickly navigate to those pages or options.
  • My Reports
    The My Reports pane on the Home page lists all reports if you have been authorized access by your administrator.
  • Top-level Menu
    The top-level menu provides links to frequently used features you might want to access while you are working and also allows you to navigate between different lines of business.
