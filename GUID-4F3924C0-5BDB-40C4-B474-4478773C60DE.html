Modifying Queues

Changes in the way in which your company works may make it necessary for you to make changes to queues, such as changing the filter criteria.

When modifying a queue, you can change the fields available to the user, as well as any sorting and filtering criteria. You can also change the assignment of users and groups for the queue.

  1. In the administration navigation pane, click Queues.
  2. (Optional) In the search area field, enter search filter criteria and click Search.
  3. Select the queue to modify and click the Edit queue icon for that row.
  4. Modify the queue details as needed.
  5. Click Save to save the queue.
