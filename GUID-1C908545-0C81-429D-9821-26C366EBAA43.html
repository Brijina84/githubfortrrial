Adding Notes to an Application in Review

If you use Synchronous Consumer Line of Business, use Notes to add information to an application during application review. Notes record information about an application. Notes can be free-form text, predefined notes, or any of the notes categories. For example, Applicant called to check status of application can be a pre-defined note that you add to an application.
You must have the ManageNotes role to add a note to an application in review or edit and delete notes. If you do not have this role, you can only view the notes that have been added to the application.
  1. Open the application that is in review and for which you want to add a Note.
    The Notes pane is at the bottom of the review page.
  2. To give a label or type to your note, select the note category from the category list, and then add the text for the note in the Note field.
  3. To add a predefined note, select the note from the list of predefined notes.
    If required, combine the predefined note with a custom note.
  4. To add a custom note, add the text in the Note field.
