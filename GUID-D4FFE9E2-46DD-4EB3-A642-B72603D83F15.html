Loan Payment Type Values

These values can appear in the following places:
  • The LoanPaymentType.Code attribute in the om_apm_product_core module in Designer.
  • The ApplicationOffer.LoanPaymentTypeId and creditApplicationProduct.loanPaymentType attributes in the om_apm_consumer module in Designer.
  • The Domain Values > LoanPaymentType page in the Application Processing Module user interface.
