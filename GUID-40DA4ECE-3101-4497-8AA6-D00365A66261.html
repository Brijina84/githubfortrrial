Document Template Keyword Values

These values can appear in the following places:
  • The UserDocumentKeywords.Name attribute in the om_apm_docgen module in Designer.
  • The Keywords_Excel.Name attribute in the om_apm_bootstrap module in Designer.
  • The Document Templates > Keywords page in the Application Processing Module user interface.
