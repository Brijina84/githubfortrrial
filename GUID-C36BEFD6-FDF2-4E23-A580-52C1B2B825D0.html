Managing Custom Functions

You can write custom functions to implement specific business logic and use them in data methods just like other functions. From the Data Method Expression Toolbar, you can access the Tools option to manage custom functions. You can upload files that contain the custom classes or functions, delete the files, and enable or disable the custom functions.
  1. Open the data method in the Edit mode.
  2. On the Data Method Expression Toolbar, click the Manage Custom Functions icon.
    The Custom Files and Functions page appears.
  3. Use the Add icon to upload a custom file and the Delete icon to delete a custom file.
  4. Enable a custom function in the file if you want to use it in a data method.
  5. Click the link to navigate back to the data method page.
