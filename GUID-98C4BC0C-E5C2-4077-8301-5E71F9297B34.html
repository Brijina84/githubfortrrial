Viewing a List of Strategy Changes

Use the View button on the Audit Strategies page to get a report of item changes that were made within a specific period.
Note The Audit Strategies option is available to users who have auditing permissions. For more information, see the FICO Origination Manager Decision Module Administrator's Guide.
  1. In the Tools tab, select Audit Strategies from the left navigation pane.
  2. On the Audit Strategies page, select a start and end date for the report.
    The current date is the default for both settings.
  3. Click View.
    Item changes in the specified period are listed. Changes that were made on the start and end dates are included.
  4. To obtain a copy of the report, click Print.
    The report is displayed in a new page. You can select a printer, styles, and other options for the report.
