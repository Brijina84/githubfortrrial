Creating a Ruleset

Use the Create button on the Rulesets list page to add a new ruleset.
  1. In the navigation pane, click Rulesets.
  2. On the Rulesets list page, click the Create button.
  3. In the Create dialog box, enter a name that is unique among rulesets.
    Names must be text strings of 150 characters or less. Names can contain letters, numbers, and underscores.
  4. (Optional) In the Description field, enter text describing the ruleset.
  5. Click Create.
    The new ruleset is listed on the Rulesets list page.

    To specify the properties for the ruleset and add rules, you need to edit the ruleset. At this point, you can proceed with the following steps to set the properties and add rules, or you can check in the ruleset to commit it to the configuration repository. If you check in the ruleset, you need to check it out when you are ready to edit the properties or add rules.

  6. Click the ruleset name on the Rulesets list page.
    The Ruleset Properties page appears.
  7. Click the Edit icon.
  8. The ruleset is created with No Decision as the default value for the decision level. To change this, select a different value from the first drop-down list for Return decision.
    The available options include Review, Verify, User 1, User 2, User 3, Re-evaluate, PossibleFraud, Decline, Recommend Decline, Investigate, Recommend Approve, Approve, and No Decision.
  9. The ruleset, by default, is set to return a decision if any of its rules fires. To have the Decision Module use a ruleset severity threshold when processing rules, perform the following steps:
    1. From the second drop-down list for Return decision, select If total rule severity exceeds.
    2. In the box that appears, enter a number for the threshold.
      The threshold number specifies the cumulative rule severity value above which the ruleset registers a decision response for the specified decision type.
  10. To add a result for the ruleset, click the Add icon for the Results field.
    A result is added with a system-generated name. You can click on the name and edit it inline.
  11. To add rules to the ruleset, follow the steps described in Adding a Rule to a Ruleset .
  12. Click Save.
