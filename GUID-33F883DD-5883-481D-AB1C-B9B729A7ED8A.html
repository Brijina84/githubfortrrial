Create Application

The Create Application option on the Home page provides a way to access the Create Application page and create applications for new or existing customers. Based on whether you have selected OM APM Consumer, OM APM Synchronous Consumer, or OM APM Small Business from the top-level menu, you can create a consumer, synchronous consumer, or small business application, respectively. Users follow data entry guidelines to enter the required data and then submit the application or save it for additional data entry at a later time.
