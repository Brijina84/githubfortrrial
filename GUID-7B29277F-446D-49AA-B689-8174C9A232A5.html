Managing Variables Overview

Perform the following steps to create variables that map to data fields in the object model:

  1. Add a variable in the properties page for an item type.
  2. Select a data type for your mapping such as Real, Integer, or String.
  3. Browse the object model to change the object or reference.
  4. Select a data field or data method that maps to the variable.
