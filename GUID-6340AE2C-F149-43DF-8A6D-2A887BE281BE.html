Decision Scenarios

A decision scenario is a mechanism for recording decisions from within an action node of a decision tree. Decision scenarios, as opposed to other types of decisioning components, are generally used when you want to:
  • Make a decision based on conditions that may be expressed in a tree form. Multiple decisions may be made for a given set of conditions (multiple decision scenarios called from a single action node).
  • Associate more than one reason with the decision, where the reason is fixed and not based on rules.
  • Reuse the specific decision and reason combination in multiple product strategies or multiple decision trees within a single decision flow.

For each decision scenario, you can specify a decision result and one or more reason codes.

When a decision scenario is executed, it creates one or more specific objects under the DecisionResponse object, including:
  • A single Decision object that is a child of the Product object associated with the current product. The Decision object is populated from the decision scenario information provided when the decision scenario is defined.
  • Zero or more Reason objects (each of which is a child of the Decision object), one for each reason that is provided when the decision scenario is defined.

If desired, other Decision object fields can be populated by executing a data method sequence in the same decision tree action node, or in a later decision flow step.

Covered Topics
  • Decision Scenario Properties
    The Decision Scenario Properties page displays the properties associated with a decision scenario, such as the type of decision to record and the reasons to be returned with the decision response.
  • Creating a Decision Scenario
    Use the Create button on the Scenarios list page to add a new decision scenario.
  • Copying a Decision Scenario
    Use the Save As icon in the Decision Scenario Properties page to add a new decision scenario based on an existing one.
  • Deleting a Decision Scenario
    Use the Delete icon on the Scenarios list page to delete a decision scenario that is no longer needed.
  • Deleting a Reason in a Decision Scenario
    Use the Remove current reason icon in the Reasons section to remove a reason specified for a decision scenario.
