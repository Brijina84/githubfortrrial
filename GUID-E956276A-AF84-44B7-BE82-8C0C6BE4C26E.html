Message Levels

The level of severity of a message is indicated by a code. This appears in the Application/MessageList/Message/@SeverityCode attribute. The entire contents of Application/MessageList/Message are also given in the Outcome/Message/MessageDetail/serviceMessage node of the envelope.
Message Severity Levels
Severity code Level
0 Information
1 Warning
2 User Error
3 System Error 
<Message MessageNumber="10002" SeverityCode="2">
    <Description>Request Type (1234) Is Not
    Supported</Description>
    <Resolution>Use a valid request type and retry the
    request.</Resolution>
</Message>

A level 2 message

Descriptive information about errors is returned in the Application/MessageList/Message/Description element. For example, if a request type is not supported, the name of the submitted request type is returned as part of the Description element.
