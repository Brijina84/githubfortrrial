Copying a Score Model

Use the Save As icon on the Score Model Properties page to add a new score model based on an existing one.
  1. On the Score Model Properties page of the score model you want to copy, click the Save As icon.
  2. In the Name field, enter a unique name for the score model.
  3. Click Save As.
    The Score Model Properties page of the new score model appears.
  4. Click Edit, and modify the description and any other properties you want to change.

    You can also add or remove variables and characteristics. When you delete a variable or a characteristic, you will be prompted to confirm the deletion.

  5. Click Save.
