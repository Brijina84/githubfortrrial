Adding an Input Parameter

Use the Add icon in the Input Parameters section to define the parameters (input variables) to be passed to a specific data method and used in its expression.
  1. In the Data Methods list page, select the data method for which you want to add an input parameter.
    The Data Method Properties page appears.
  2. Make sure you can edit the data method:
    • If the data method has been checked in, click Check Out.
    • If this is a new data method that has not been checked in yet, click Edit.
  3. In the Input Parameters section, click the Add icon.
    An input parameter with default values for Name (xInputs1, xInputs2, and so on) and Type (string) is added.
  4. To change the default properties, click the parameter's name to display the Input Parameter page.
    1. Enter a name (unique to input parameters) in the Name field.
      Certain lowercase words are Blaze Advisor reserved words and cannot be used in names. For a list of these words, see Avoiding Name Conflicts. You can avoid this problem by using mixed-case names.

      The system validates the name and replaces or deletes invalid characters.

    2. Select a data or object type from the list in the Type field.
  5. Click Save.
    You can use this input parameter in the data methodâ€™s expression.
Tip If you want to add another input parameter, use the breadcrumbs path at the top of the page to get back to the Data Method Properties page.

To change the position of a parameter in the list of inputs, use the Move Up and Move Down arrow icons in the Input Parameters section.

Note If one data method invokes another data method with inputs, it must send the inputs in the exact order defined.
