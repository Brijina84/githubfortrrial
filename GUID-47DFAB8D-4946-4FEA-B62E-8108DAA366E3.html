Expression Editor Color Coding

The Expression Editor provides color coding and formatting of text according to its use. The following table contains a list of items, color coding, and formatting conventions used.
Color Codes
Text Color
Data Method Expression Language keywords (such as for, every, if, then) Gray
Functions, data methods, and data object references Green
Reserved values (such as null, false, true) Dark orange
Numbers and operators Black
Strings within quotes ("") Pink
Local and input variables Purple
Comments Gray (italics)
Data objects and fields Blue
