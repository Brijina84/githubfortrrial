Creating a Scoring Scenario

Use the Create button on the Scenarios list page to add a new scoring scenario.
  1. In the navigation pane, click Scenarios.
  2. On the Scenarios list page, click the Create button.
  3. In the Create dialog box, enter a unique name.
    Names must be text strings and can contain alphabets, numbers, and underscores.
  4. In the Description field, enter text describing the scoring scenario.
  5. In the Type drop-down list, select Scoring Scenario.
  6. Click Create.
    The new scoring scenario is listed on the Scenarios list page.

    To specify the properties for the scoring scenario, you need to edit the scenario. At this point, you can proceed with the following steps to set the properties, or you can check in the scoring scenario to commit it to the configuration repository. If you check in the scenario, you need to check it out when you are ready to edit the properties.

  7. Click the scoring scenario name on the Scenarios list page.
    The Scoring Scenario Properties page appears.
  8. Click the Edit icon.
  9. For each score model row you want to add, perform the following steps in the Score Model Information section.
    1. Click the Add a new score condition icon.
    2. In the new row that is displayed, select a score model from the Name drop-down list.
    3. In the Score > = field, enter a non-negative integer specifying the minimum score value required to register a decision.
    4. In the and < field, enter a non-negative integer specifying the maximum score value required to register a decision.
      This value must be greater than or equal to the minimum score value you specified.
    5. In the Decision drop-down list, select the decision that will be registered as a candidate decision if the returned score is between the lower and upper cutoffs (inclusive).
  10. Click Save.
