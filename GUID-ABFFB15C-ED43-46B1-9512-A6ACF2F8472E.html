Withdrawing Applications

When an application has to be withdrawn (for example, when a credit applicant elects to do so), you can locate the application and withdraw it from the system.
Applications in any of the following states may be withdrawn:
  • Draft
  • Submitted
  • Pending Duplicate Resolution
  • Pending Review Resolution
  • Pending Verification

Perform the following steps to withdraw an application:

  1. Locate and open the application.
  2. In the Application page, review information as needed.
  3. Click Withdraw Application.
  4. In the Withdraw Reason field, select a reason for the withdrawal.
  5. Click Yes.
    The system sets the status of the application to Pending Withdrawal Archive.
