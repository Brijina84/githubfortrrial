Test Icons

Use the icons in the Test tab to manage base applications and test scenarios.
The following table describes the icons in the Base Applications list page and Test Scenarios list page:
Name Icon Description
Delete Deletes a base application or test scenario from the repository.
Import base application Imports a base application in the XML format to the repository.
Duplicate selected item Creates a copy of the base application or test scenario with a different name.
