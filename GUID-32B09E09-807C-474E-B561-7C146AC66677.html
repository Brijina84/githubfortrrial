Strategy Component Snapshot Icon Reference

The icon reference table explains the icon graphics that appear on the Strategy Component Snapshot page and report.
Icon Name Description
Open calender Selects a date for the report.
Version Displays the latest version number of the item. "New" indicates that the item exits only in the private workspace of the user and has not been checked into the configuration repository.
Check-in date Displays the date when the item was last checked in.
Check-in comment Displays the comment that was entered when the item was last checked in.
