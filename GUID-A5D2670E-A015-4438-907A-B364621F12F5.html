Example: Map Variable to Data Field

You can map a variable called MonthlyIncome to the data field Application.creditApplication.personalApplicant.applicantFinancial.income.grossMonthlyIncome in the object model.
  1. Open the properties page of an item that belongs to one of the following item types: score model, decision tree, decision table, or product strategy.
  2. In the Manage Variables section, click the Add icon.
    A variable appears in the Manage Variables section. You can select the variable and modify the default field values.
  3. Select the variable to modify the variable properties.
    The Variable Properties page appears.
  4. Modify the name of the variable to MonthlyIncome.
  5. Optionally, add the following description: The variable MontlyIncome maps to a data field in the object model.
  6. Modify the data type for the variable in the Type list. Select Real from the list.
  7. Select the Change icon to modify the field that the variable maps to in the object model.
    The object model browser appears.
  8. In the object model browser, select the Object or Reference by expanding the object model.
  9. Expand the object model and select Application>creditApplication>personalApplicant>applicantFinancial income.
    You can also search for applicantFinancial in the object model. The Decision Module sets the Object or Reference field with the path of the object you selected.
    The Select Field or Data Method section shows the fields that are of data type real.
  10. Select the field grossMonthlyIncome.
  11. Click Select.
    The variable MonthlyIncome maps to a data field that gets data from the XML transaction input.
