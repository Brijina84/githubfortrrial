Expression Editor Toolbar Icons

The Expression Editor toolbar contains icons for inserting operations, functions, and object model items into data method expressions, and for validating data method expression statements.
The following table lists the toolbar icons and their actions.
Expression Editor Toolbar Icons
Icon Action
Insert an operation
Insert a function
Insert an item from the object model
Validate data method code
Insert an Excel file in ExcelCellValue function
Manage Custom Functions. Clicking the icon takes you to the Custom Files and Functions page in the Tools tab.
Manage Lookup Values. Clicking the icon takes you to the Excel Lookup Values page in the Tools tab.

Within the Expression Editor, pressing Ctrl + t shifts focus to the first icon in the toolbar. You can press Enter to select this icon and Tab to focus on the next icon.
