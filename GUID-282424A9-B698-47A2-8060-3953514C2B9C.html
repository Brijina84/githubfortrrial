Version Comparison

On an item's properties page, the History option in the Version drop-down list displays two tables—one containing information about the working copy, and the other listing all previous versions of the item. In the Version History table, you can select any two versions of the item and compare them.
Covered Topics
  • Comparing Item Versions
    Use the Compare option for the Version History table to compare two versions of an item.
  • The Visual Differences Window
    The Visual Differences window displays the results of comparing two versions of an item.
  • Comparison Results for Rulesets and Rules
    When you compare two versions of a ruleset, the differences in the properties of the ruleset and its rules are highlighted.
  • Comparison Results for Categories and Products
    When you compare two versions of a category, the differences in the properties of the category, and the names of its products and product selection trees are highlighted. When you compare two versions of a product, the differences in its properties and promotion values are highlighted.
  • Color Key for Highlighting Differences
    The Visual Differences window highlights the differences between two item versions using a color key.
