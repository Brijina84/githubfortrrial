Creating a Decision Flow

Use the Create button in the Decision Flows list page to create a decision flow.
  1. In the Decision Flows list page, click the Create button.
    The Create dialog box appears.
  2. Enter a unique name for the decision flow in the Name field .
    Names must be text strings of 150 characters or less. Names can contain letters, numbers, and underscores.
  3. Optionally, enter a description in the Description field.
  4. Click Create.
    The new decision flow appears in the decision flow list page.
  5. Select the new decision flow to open the Decision Flow Properties page.
  6. Optionally, click the Edit icon and select the from and to arrows to enter active dates for the decision flow.

    If you do not set a Active from date, the decision flow is active as soon as it is submitted and promoted into production. If you do not set a to date, the decision flow is effective indefinitely.

  7. Click Save.
  8. Click Check In to commit the decision flow to the configuration repository.
After you create and check in a decision flow, to add and manage decision flow steps, you must check out the decision flow.
