CRA Scoring Model Response Values

Output from a consumer credit report request contains information on the scoring model used to produce online credit bureau score 1 or online credit bureau score 2. These values are used by the Data Acquisition Module and Analytic Module adapters.

The Value column contains a FICO-specific scoring model identifier. In the Score 1 column, an X denotes a scoring model that is acceptable as score 1 for transactions using the ARM 3.0 model suite.
