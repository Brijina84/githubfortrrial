Creating a Base Application

Use the Create button in the Base Applications list page to create a new base application.
  1. Select the Test tab in the SMW.
  2. Click Base Applications.
    The Base Applications list page appears.
  3. Click the Create button.
    The Add Base Application page appears.
  4. Enter a unique name for the base application.
    The base application name can contain letters, numbers, underscore and the following special characters: ', -
  5. Optionally, enter a description in the Description field.
  6. Enter a test application in XML format in the Application XML window.
    You must enter input application XML to create the base application.
  7. Click Create.
    The base application appears in the Base Applications list page.
  8. Alternatively, import an application from the Base Applications list page.
    You can browse and import a standard input application in a .xml file format. The SMW assigns a unique name for the application.
    Note If you are using Internet Explorer 9, you can only import one base application at a time due to a limitation in IE 9.
