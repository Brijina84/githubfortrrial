Modifying Document Templates

Use the Document Templates > Edit <Template Name> page to modify templates.
  1. In the administration navigation pane, click Document Templates > Templates.
  2. (Optional) In the search area field, enter search filter criteria and click Search.
  3. Select the document template to modify and click the Edit icon for that row.
    Note If two users try to edit a document template at the same time, changes are only accepted for the first user to click Save.
  4. Select Download current template. Make the desired modifications to the template.
  5. Browse and select a document template in .docx format from your filesystem.
  6. When you are finished making your changes, click Publish or Save.
    Note Clicking Save only save the changes made to the document template. The workflow will not use the changes made to the document template until it is published. By clicking Publish, the document template changes are made available in the workflow.
