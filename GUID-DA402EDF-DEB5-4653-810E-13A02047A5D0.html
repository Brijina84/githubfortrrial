Managing Rules

Use the options on the properties page for a ruleset to add and remove rules. Use the options on the properties page for a rule to define its outcome settings, scope, and conditions.
Covered Topics
  • Adding a Rule to a Ruleset
    Use the Add icon in the Rules section of the Ruleset Properties page to add rules to a ruleset.
  • Setting the Scope of a Rule
    Use the lists for the Return outcome when property to set the scope of a rule.
  • Adding and Configuring a Rule Condition
    Use the Add icon in the Rule definition section of the Rule Properties page to add and configure rule conditions.
  • Removing a Rule from a Ruleset
    Use the Delete icon in the Rules section of the Ruleset Properties page to remove a rule from its associated ruleset.
