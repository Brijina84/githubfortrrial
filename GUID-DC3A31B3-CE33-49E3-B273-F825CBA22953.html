Configuring Java Class Files in Blaze Advisor

Add the Java Class files and set the Java Classpath in Blaze Advisor
  1. Add the JAR file to the Blaze Advisor Java Classpath in the following file:<BlazeAdvisor_Home>\bin\setenv.bat
  2. Connect to the EOS project in Blaze Advisor.
  3. Create a folder in the repository to save the new Java Business Object Model (BOM).
  4. Select Project > Business Object Models > New Java Business Object Model
    Java Business Object Model Wizard opens.
  5. Add the Java class files that you want to access from the JAR file.
  6. Now, you can create a Blaze Advisor function that calls a Java program which can access the Java classes.
    Note For more information on creating a Blaze Advisor function and using InvokeBlazeFunction, see FICO Origination Manager Decision Module User's Guide
  7. Add the JAR file to the Web application server directory before you use the function in application processing.
    For example:
    • Tomcat: c:\programFiles\ApacheSoftwareFoundation\Tomcat7.0\webapps\SMW.sysid\WEB-INF\lib and \webapps\ps\web-inf\lib
    • WebSphere Application Server (WAS): installedapps\SMW.war\SMW.sysid\web-inf\lib and installedapps\ps.war\web-inf\lib
