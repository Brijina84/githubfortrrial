Queue Summary Report

The Queue Summary Report allows managers to know how much work is in each queue and to compare workloads across all queues. Use it to see how many users and user groups are assigned to a queue, and the total number of applications in the queue. This is a runtime report and is not driven by date parameters that you select.
Queue Summary Report Fields
Field Name Description
Queue Name The name of the queue.
Queue Description A description of the queue.
Users The number of users assigned to the queue.
Groups The number of workgroups assigned to the queue.
Applications Per Queue The total number of applications currently in the queue.
