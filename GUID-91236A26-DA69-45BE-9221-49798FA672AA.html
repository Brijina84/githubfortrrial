Functions

Data method expression statements frequently contain functions. A function is a named procedure that performs a specific task. The statement that requests the function is called a function call.
The following functions are available in the SMW and can be used in creating new data methods or modifying existing ones.
Data Method Functions
Function Type Name Description
Object Attach Attaches one object to another.
Detach Detaches one object from another.
