Entity Properties

An entity is any object that you want in your domain model and store information about it. You can identify an entity by the plus (+) sign next to its name. You can click the plus sign to expand the view and see the fields under that entity.

Add an entity by clicking the Add Entity symbol.

Note You can add an entity only to another entity.
An entity has the following properties:
Entity Properties 
Name
 
Description
 
Type
 
Mandatory
Name Name for the entity. String Yes
Description Description to help understand why the entity is used. String Yes
Display name Name displayed for the entity in the UI. Yes
