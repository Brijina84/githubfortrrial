TimestampFromStr

Syntax: TimestampFromStr(string)

Description: Converts a string to a timestamp, for use in timestamp operations.

Inputs:

  • string: A string of the form “yyyy-MM-dd HH:mm:ss:S”, where “yyyy-MM-dd” is an optional date string, and “HH:mm:ss.S” is an optional time string.

Return Type: timestamp

Returns:

  • The timestamp whose value corresponds to the date/time provided in the string.
  • If string does not include a date portion, use “1199-11-11” for the date portion of the timestamp.
  • If string does not include a time portion, use midnight (00:00:00) for the time portion of the timestamp.
  • If string is null, the function returns null and also throws an exception.
  • If string is not a valid string, the function throws an exception.
  • If string is an empty or zero-length string, the function returns null.

Example: 

localTimestamp := TimestampFromStr(“2005-12-16 12:00:00.0”);
