Checking In a Product

Use the Check In icon from Product Properties page to check in a product.
  1. In the Category List page, click the category containing the desired product.
  2. In the product list, click the product.
    The Product Properties page appears.
  3. Click the Check In icon.
    You will be prompted for check-in comments.
  4. Enter comments in the Check-in Comments dialog box. Click Check In.
    Note
    The following rules apply when you check in products:
    • If you create a category and add products, the products are checked in when you check in the category. Once you check in a category, you can independantly manage the products from the Product Properties page.
    • If you create a category and add products, the category is checked in when you check in a product
    • If you add a product to an existing category, then you can navigate to the Product Properties page to check in the product. You must check out and edit a category to add products.
    For more information on checking in an item, see also Checking In an Item.
