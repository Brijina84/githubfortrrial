XPath Expressions for Input Test Attributes

You can enter input test attributes as XPath expressions to create test scenarios.

The following examples show valid formats for XPath expressions when you enter input test attributes:

/Application/creditApplication[/Application/creditApplication/personalApplicant/applicantFinancial/outTotalMonthlyPayments]

/Application/creditApplication[personalApplicant/applicantFinancial/outTotalMontlyPayments]

/Application/CreditRequest[@DMFunction="FR"]

/Application/CreditRequest[@DMFunction]

Note Some more complex XPath expression formats are not supported for input test attributes. You must use XPath expressions that conform to the valid formats.
