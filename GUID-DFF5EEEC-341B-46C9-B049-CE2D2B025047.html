Business Processes Designer

The Business Processes workbench provides a designer for creating a business process or a workflow.
The designer has the following sections:
  • Tools or elements available to construct the workflows on the left hand side
  • Work canvas or grid in the center
  • Properties section where you specify the properties and parameters that need to be passed to and from the elements or tools, when they are used in a construct, on the right-most side
The toolbar of the designer contains the following tools:
Business Process Canvas 
Icon
 
Icon Name
 
Description
Cut Deletes an element, but makes it available for pasting on the canvas.
Copy Copies an element, retains the original element on the canvas, and makes it available for pasting it on the canvas.
Paste Pastes an element on the canvas.
Delete Deletes the selected element.
Zoom-in Expands the scale and lets you see detailed information on the canvas.
Zoom-out Contracts the elements on the canvas to fit on the canvas.
Fit-to-canvas Fits all elements to the canvas automatically.
Grid Shows and hides the grid. This icon works as a toggle.
Download model png Downloads as a PNG snapshot the elements and the connections on the canvas.
Show comment Opens a window on the right side of the canvas to show any comments that have been added.
Create comment Opens a window on the right hand side of the canvas to enable you to add a comment.
