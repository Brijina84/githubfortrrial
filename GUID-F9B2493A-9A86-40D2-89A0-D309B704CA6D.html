Implementing a Custom Function

To make the Decision Module aware of a custom function, you need to perform the following tasks.
  1. Implement the IDAJavaExtensionDispatcher interface to provide the name, arguments, return type, and description of your custom function.
  2. Create a JAR file containing the details of your implementation.
