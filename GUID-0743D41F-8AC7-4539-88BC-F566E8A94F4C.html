Pricing Scenarios Properties

The Pricing Scenario Properties page displays the properties associated with a pricing scenario, such as the interest rate and the amount of a product offering.

The Pricing Scenario Properties page is displayed when you click the name of a pricing scenario on the Scenarios list page. You can specify the properties when you check out the scenario and click the Edit icon.

Pricing Scenario Properties
Property Description
Name A unique name that can contain only alphabets, numbers, and underscores.
Description The purpose or work of the pricing scenario (optional).
Offer ID An alphanumeric identifier for the offer.
Rate (%) A real number specifying the interest rate percentage of the offer.
Amount An integer specifying the amount of the offer.
Term (months) An integer specifying the number of months in the product offer loan term.
