Score Model Reports

Score models generate characteristic data for reporting.
To assist you in monitoring your models, the Decision Module provides the following scoring reports:
  • Characteristic Analysis—Shows information useful in analyzing score model characteristics for score models defined and executed within the Decision Module.
  • Population Stability—Shows information useful in analyzing the stability of a score model, relative to its expected performance across score ranges. This report also includes a stability index, which is an overall measure of the stability of the score model.
  • Final Score—Shows information useful in analyzing a scoring scenario’s approval rate, and whether there are inconsistencies that require further investigation.

When requesting scoring reports, you can select information by date, system, product, and scoring scenario. You specify these criteria using the report request procedures described in the FICO Origination Manager Decision Module Administrator's Guide.
