Product Selection Tree Tasks

You can perform tasks, such as insert, copy, edit, clear, assign work on a selected tree node.

Click on the start node, condition split branch node or action node to see the commands for an element in the Edit menu.

Product Selection Tree Edit Commands
Node Commands
Start Node If you have just created the product selection tree when you select the Start node you see the following commands:
  • Insert split
  • Assign expression
  • Insert level

If you select a Start node with at least one level and split, you see the following commands:

  • Collapse subtree
  • Expand subtree (expands only the first level of the tree)
  • Edit split
  • Insert split
  • Cut subtree
  • Copy subtree
  • Clear subtree
Branch Node If you select a condition split branch node, you see the following commands:
  • Collapse subtree
  • Collapse entire tree
  • Expand subtree (expands the entire tree)
  • Edit split
  • Insert split
  • Cut subtree
  • Copy subtree
  • Clear subtree
  • Paste action/subtree*
Note *The Paste action/subtree command is displayed and enabled only if you have copied a subtree.
Branch Node on the last level If you select a condition split branch node where no additional splits are needed, you see the following commands:
  • Edit expression
  • Copy action
  • Clear action
Action Node If you select an action node, you see the Assign action command that you can use to edit your action.
