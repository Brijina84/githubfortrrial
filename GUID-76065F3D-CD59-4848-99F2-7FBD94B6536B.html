Locating and Viewing Applications

There are four methods of locating and viewing applications.
  • Select an application from a queue.
  • Use the Search Application feature.
  • Use the Open Application Number feature.
  • Use the Next Case option when you want to open and view the next application in the list.
Covered Topics
  • Selecting an Application from a Queue
    Use the My Queues pane on the Home page to view and access the queues assigned to you through a user, or assigned to a group of which you are a member.
  • Searching for an Application
    Use the Search Application feature to search for a specific application
  • Opening an Application
    When you are working on an application and want to open a different application, use the Open application number field that appears in the upper right corner of the page.
