Deleting Domain Values

Administrators with the proper permissions can delete domain value information from the system.

Note The system does not prevent domain values that are in use from being removed. If you delete a domain value that is in use, it will still appear within an application; however, you will not be able to use this domain value in newly created or edited applications.
  1. In the administration navigation pane, click Domain Values and then click the Edit icon in the row for which you want to remove the domain value.
  2. (Optional) In the search area field, enter search filter criteria and click Search.
  3. Select the domain value to delete and click the Delete <Domain Value Name> icon for that row.
  4. Click OK in the confirmation dialog box that appears.
